# 기본 기계요소 6가지 (★★★)

#mechanism

## Overview Table (한눈에 비교)

| 기계요소 | 입력 | 출력 | 프로젝트 활용도 |
|----------|------|------|----------------|
| 지레 (Lever) | 힘 | 증폭된 힘 / 방향 전환 | 링크장치의 기본 단위 ★★★ |
| 도르래 (Pulley) | 힘 | 방향 전환 / 기계적 이득 | 와이어 구동, 무게 경감 ★★ |

## 1. 지레 (Lever)

지레는 받침점(fulcrum)을 기준으로 힘을 증폭하거나 방향을 바꾼다.

**3종류 지레:**
```
1종 지레: [힘] —[받침]— [하중]   → 방향 전환 + 증폭 가능 (시소, 가위)
2종 지레: [힘] ———— [하중][받침] → 증폭만 가능 (외바퀴 손수레)
3종 지레: [받침] ——[힘]—— [하중] → 속도 증폭 (핀셋, 낚싯대)
```

> [!tip] 프로젝트 적용
> 4절 링크의 각 링크가 곧 지레 역할 → [[4.4절 링크장치]]

**기계적 이득(MA):** `MA = 힘 팔 / 하중 팔`

---

## 2. 도르래 (Pulley)

- **고정 도르래**: 힘 방향만 전환, 이득 없음
- **움직 도르래**: MA = 2 (힘 절반)
- **복합 도르래**: MA = 도르래 개수

> [!tip] 프로젝트 적용
> 와이어+도르래로 말 다리를 원격 구동 시 유용. 로프 방향 자유롭게 바꿀 수 있음.

---

## 3. 바퀴와 축 (Wheel & Axle)

회전력(토크) 증폭의 핵심 원리. 기어와 동일한 메커니즘.

```
MA = R_wheel / R_axle
```

큰 바퀴 → 작은 축: 토크 증폭 (속도 감소)
작은 바퀴 → 큰 축: 속도 증폭 (토크 감소)

---

## 4. 경사면 & 쐐기 (Inclined Plane & Wedge)

캠(Cam)의 기본 원리. 경사면을 따라 이동하면 수직 방향 힘 발생.

```
MA = 빗면 길이 / 높이 = 1 / sin(θ)   ← 마찰 무시 이상값
```

> [!important] 캠과의 연결
> 캠의 로브(lobe) = 경사면의 회전 버전. 회전 입력 → 직선 출력 → [[3.캠과 팔로워]]

---

## 5. 나사 (Screw)

나사는 경사면을 나선형으로 감은 것. 회전 운동 → 직선 운동.

```
이동거리 (1회전당) = 리드(lead) = 피치 × 나선 수
```

**리드스크류(Lead Screw):** 모터 회전 → 정밀 직선 이동. 자기 잠금(self-locking) 특성.

---

## 6. 기어 (Gear)

이빨이 있는 바퀴. 미끄럼 없는 동력 전달.

→ [[5.기어와 동력전달]] 에서 상세히

## Exam/Test Patterns (시험 빈출 패턴)

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "힘을 줄이고 속도를 높이려면" | 3종 지레 또는 소→대 기어비 |
| "토크를 높이고 속도를 줄이려면" | 2종 지레 또는 대→소 기어비 |
| "회전 → 직선 운동" | 크랭크, 캠, 나사(리드스크류) |
| "자기 잠금 필요" | 나사(screw), 웜기어 |
| 캠의 기본 원리 | 경사면(inclined plane) |
