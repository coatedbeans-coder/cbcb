# 크랭크와 왕복운동 (Crank & Reciprocating Motion) (★★★)

#mechanism #linkage

## Overview Table

| 항목 | 설명 |
|------|------|
| **크랭크 (Crank)** | 회전축에 연결된 레버. 회전 → 왕복 변환의 핵심 |
| **스로우 (Throw)** | 크랭크 핀이 그리는 원의 직경 = 왕복 운동 거리 |
| **크랭크-슬라이더** | 크랭크 + 커넥팅 로드 + 슬라이더로 구성된 기본 메커니즘 |
| **왕복운동 (Reciprocating)** | 앞뒤 또는 상하로 반복되는 직선 운동 |

---

## 크랭크의 두 가지 역할

```
역할 1: 손잡이로 사용 (사람이 크랭크를 돌림)
  → 입력: 회전력(토크)   출력: 축 회전
  → 예: 우물 두레박 핸들, 자전거 페달

역할 2: 모터가 크랭크를 구동 (회전 → 왕복)
  → 입력: 모터 회전    출력: 왕복 직선 운동
  → 예: 자동차 엔진 피스톤, 재봉틀 바늘
```

---

## 크랭크-슬라이더 메커니즘

가장 기본적인 회전→직선 변환 메커니즘.

```
     크랭크 핀 (r)
         ●─────────●──────────●
         │커넥팅 로드│          슬라이더
    ●────┤          │          ═══════→ 왕복
  회전축  ●─────────●
         
  스트로크 = 2r (크랭크 반지름의 2배)
```

**특성:**
- 크랭크가 등속 회전해도 슬라이더 속도는 불균일 (정현파 형태)
- 상사점(TDC)·하사점(BDC) 근처에서 느리고, 중간에서 빠름

## 말 다리 구동에 크랭크 적용

> [!important] 다리 설계 전략
>
> 크랭크-슬라이더 대신 **크랭크-로커** 구성 권장:
>
> ```
>    모터축 ─●─ 크랭크 ─●─ 커넥팅로드 ─●─ 로커(다리)
>                r              L              
>   
>   다리 스윙 각도 = arcsin(r/D) × 2  (근사, 크랭크 최대 회전 기준)
> ```
>
> - r (크랭크 반지름): 35mm
> - L (커넥팅 로드): 100mm  ← 공식 분모가 아님, 위상 전달용
> - D (다리 길이, 어깨 피봇~발굽): 150mm  ← 공식 분모
> - 다리 스윙 각도: 약 **10° 단방향** (θ = arcsin(r·sin45°/D))
>   → 서보 양방향 ±45° 구동 시 총 스윙 ≈ 20°

**위상 제어:**
```
4다리 크랭크를 하나의 캠샤프트에 (올바른 갤럽 순서):
  뒤우(우뒤): 0°
  뒤좌(좌뒤): 90°
  앞우(우앞): 180°
  앞좌(좌앞): 270°
→ 캠 또는 크랭크 핀을 90° 간격으로 배치
```

## 크랭크 제작 방법

| 방법 | 재료 | 난이도 |
|------|------|--------|
| 목재 + 나무 다월 | 합판 + 원형 봉 | ★☆☆ 쉬움 |
| 철사 구부리기 | 두꺼운 철사(2~3mm) | ★★☆ 중간 |
| CNC/레이저 | 알루미늄/아크릴 | ★★★ 정밀 |
| 기성품 크랭크 | 모형 비행기 부품 | ★☆☆ 쉬움 |

> [!tip] 쉬운 방법
> RC 비행기용 조종 혼(control horn)이나 서보 혼을 크랭크로 재활용 가능. 알루미늄이라 가볍고 강하며 나사 구멍 있음.

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "크랭크 스트로크 = ?" | `2 × 크랭크 반지름 (r)` |
| "슬라이더가 가장 빠른 위치" | 크랭크가 90° / 270° 위치일 때 |
| "회전 → 직선 가장 단순한 방법" | 크랭크-슬라이더 |
| "4다리 동시 구동, 위상 차이" | 크랭크 핀을 90° 간격으로 배치 |
| "자기 잠금 필요 없음, 단순 구조" | 크랭크 (나사와 달리 잠금 없음) |
