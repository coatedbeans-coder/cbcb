# 4절 링크장치 (Four-Bar Linkage) (★★★)

#linkage #mechanism

## Overview Table

| 링크 이름 | 역할 | 특징 |
|-----------|------|------|
| **Ground (프레임)** | 고정 링크 | 구조물에 고정됨 |
| **Crank (크랭크)** | 구동 링크 | 모터 연결. 완전 회전 or 왕복 |
| **Coupler (커플러)** | 중간 연결 링크 | 가장 흥미로운 운동 경로 생성 |
| **Follower (팔로워/로커)** | 출력 링크 | 크랭크 반대편. 완전 회전 or 로커 |

---

## Grashof 법칙 (가장 중요!)

> [!important] Grashof의 법칙
> `s + l < p + q` 이면 "Grashof 링크장치"
>
> - s = 가장 짧은 링크
> - l = 가장 긴 링크
> - p, q = 나머지 두 링크
>
> **Grashof 조건 만족 시**: 최단 링크가 크랭크(완전 360° 회전) 가능
> **조건 불만족 시**: 모든 링크가 로커(왕복만 가능)

> [!warning] 경계 케이스 주의
> `s + l = p + q` 인 경우: Grashof 경계선 → 특정 위치에서 **Lock-up(잠김)** 위험.
> 설계 시 이 조건은 반드시 회피할 것.

```
예시:
  링크 길이: s=30, l=100, p=70, q=80
  s + l = 130, p + q = 150
  130 < 150 → ✅ Grashof → 크랭크 회전 가능!
```

## 4절 링크의 3가지 형태 (Grashof 조건 만족 시)

| 형태 | 최단 링크 위치 | 동작 |
|------|---------------|------|
| **크랭크-로커** | 최단 링크 = **크랭크** (또는 팔로워) | 크랭크 완전 회전 + 팔로워 왕복 |
| **더블 크랭크** | 최단 링크 = **프레임(ground)** | 크랭크·팔로워 모두 완전 회전 |
| **더블 로커** | 최단 링크 = **커플러** | 크랭크·팔로워 모두 왕복 |

> [!tip] 말 다리에 최적
> **크랭크-로커** 형태: 모터(크랭크) 회전 → 다리(로커) 자연스러운 왕복 스윙

---

## 크랭크-로커 메커니즘 다이어그램

```
    고정 프레임 (Ground)
    A●─────────────────────●D
     │ 크랭크(Crank)         │ 팔로워/로커
     │       커플러          │
    B●─────────────────────●C
     
  모터 → A축 회전 → B가 원 궤적
  → C-D가 왕복 스윙 (로커 운동)
  
  AD = Ground(프레임), AB = Crank, BC = Coupler, CD = Rocker
```

---

## 커플러 곡선 (Coupler Curve)

커플러 중간의 한 점이 그리는 궤적. 매우 다양한 곡선 생성 가능.

- 말의 발굽 궤적(타원형) 설계에 활용 가능
- 커플러 점 위치를 조정해 자연스러운 발걸음 궤적 근사 가능

> [!tip] Cornell Kinematics 시뮬레이터
> 온라인에서 링크 길이 입력 → 커플러 곡선 미리 확인 가능
> → 설계 전 반드시 시뮬레이션 권장

## 말 다리 설계 예시

```
프로젝트: 앞다리 스윙 (크랭크-로커)

  ❌ 불만족 예시:
  Ground=150, Crank=30, Coupler=60, Rocker=70
  s=30, l=150, p=60, q=70
  s + l = 180, p + q = 130
  180 > 130 → ❌ Grashof 불만족! (모든 링크 로커 → 크랭크 완전 회전 불가)

  ✅ 만족 예시:
  Ground=150, Crank=30, Coupler=120, Rocker=80
  s=30, l=150, p=120, q=80
  s + l = 180, p + q = 200
  180 < 200 → ✅ Grashof 만족! (크랭크 완전 회전 가능)
```

## 기타 유명 링크장치

| 이름 | 특징 | 활용 |
|------|------|------|
| 팬터그래프 | 확대/축소 복사 | 스케일 조정 |
| 가위 링크 | 수직 리프트 | 테이블 리프트 |
| 워트 링크 | 직선 근사 출력 | 직선 가이드 없이 직선 운동 |
| 윈드실드 와이퍼 | 크랭크-로커 | 가장 흔한 실제 예 |

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "크랭크가 완전 회전하려면" | Grashof 조건 만족 + 최단 링크가 크랭크 |
| "모든 링크가 왕복 운동" | Grashof 조건 불만족 |
| s + l < p + q 확인 | 링크 길이 4개 나열 후 최단+최장 vs 나머지 둘 비교 |
| "다리 스윙에 적합한 링크장치" | 크랭크-로커 (crank-rocker) |
