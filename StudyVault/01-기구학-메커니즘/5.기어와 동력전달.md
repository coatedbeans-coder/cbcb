# 기어와 동력전달 (Gears & Power Transmission) (★★★)

#gear #mechanism

## Overview Table

| 전달 방식 | 장점 | 단점 | 프로젝트 적합도 |
|-----------|------|------|----------------|
| **기어 (Gear)** | 정확한 속도비, 미끄럼 없음 | 소음, 백래시(backlash) | ★★★ |
| **벨트 (Belt)** | 부드러움, 진동 흡수 | 미끄럼 가능, 장력 필요 | ★★☆ |
| **체인 (Chain)** | 미끄럼 없음, 대거리 전달 | 소음, 윤활 필요 | ★★☆ |
| **직접 결합** | 단순 | 속도비 1:1 고정 | ★☆☆ |

## 기어비 (Gear Ratio)

```
GR = N_driven / N_driver = 피동 기어 이 수 / 구동 기어 이 수

출력 회전수: ω_out = ω_in / GR
출력 토크:   τ_out = τ_in × GR  (이상적, 손실 무시)
```

> [!important] 핵심 원리
> 기어비가 크면 → **속도 감소, 토크 증가** (감속기)
> 기어비가 작으면 → **속도 증가, 토크 감소** (증속기)
> 말 구동용: 고토크 필요 → 감속 기어박스 사용 권장

**예시:**
```
모터 3000 RPM, 토크 0.1 N·m
기어비 10:1 감속기 사용 시:
  출력 속도 = 3000 / 10 = 300 RPM
  출력 토크 = 0.1 × 10 = 1.0 N·m
```

## 기어 종류

### 스퍼 기어 (Spur Gear) ★★★
가장 일반적. 평행 축 간 동력 전달.

```
  ┌──┐ ┌──┐
  │A │ │B │   A(구동) → B(피동)
  └──┘ └──┘   두 기어 맞물림, 방향 반전
```

### 헬리컬 기어 (Helical Gear)
나선형 이빨 → 스퍼보다 조용하고 부드러움. 같은 목적, 더 나은 성능.

### 웜 기어 (Worm Gear) ★★
웜(나선) + 웜 휠 조합. 직각 축 전달. 

> [!important] 웜 기어의 특성
> - 자기 잠금(self-locking): 역방향 구동 불가 → 전원 꺼도 위치 유지
> - 기어비 10:1 ~ 100:1 가능 → 매우 큰 감속비
> - 효율은 낮음 (약 40~90%, 리드각이 작을수록 낮아짐)
> - **라이더의 자세 고정에 유리** — 전원 없어도 위치 잠금

### 베벨 기어 (Bevel Gear)
직각 방향 축 전달 (90° 방향 전환).

```
  ↓ 수직 입력       → 수평 출력
```

### 래크 & 피니언 (Rack & Pinion)
회전 → 직선 변환. 기어(피니언)가 직선 래크 위를 구름.

## 벨트 & 풀리

```
속도비 = d_driver / d_driven  (지름 비)

큰 풀리 → 작은 풀리: 속도 증가, 토크 감소
작은 풀리 → 큰 풀리: 속도 감소, 토크 증가
```

**벨트 종류:**

| 종류 | 특성 |
|------|------|
| 평 벨트(Flat) | 부드러움. 미끄럼 있음 |
| V 벨트 | 마찰력 큼. 미끄럼 적음 |
| 타이밍 벨트(Timing) | 이빨 있음. 미끄럼 없음 ★ 권장 |

> [!tip] 타이밍 벨트 + 타이밍 풀리
> 3D 프린터·CNC에서 사용. 미끄럼 없고 기어비 정확. 쉽게 구매 가능.

## 감속기 선택 가이드

| 필요 감속비 | 권장 방식 |
|-------------|-----------|
| 2:1 ~ 5:1 | 기어 1단 또는 벨트 |
| 10:1 ~ 50:1 | 기어박스(2단) 또는 웜 기어 |
| 50:1 이상 | 웜 기어 또는 유성 기어박스 |

> [!tip] 프로젝트 권장
> 서보 모터는 내장 기어박스 포함 → 별도 감속기 불필요
> DC 모터 사용 시 기어드 DC 모터(Geared DC Motor) 선택으로 해결

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "속도 낮추고 토크 높이려면" | GR > 1 (감속). 피동 이 수 > 구동 이 수 |
| "역방향 구동 방지" | 웜 기어 (자기 잠금) |
| "직각 방향 전달" | 베벨 기어 or 웜 기어 |
| "미끄럼 없는 벨트" | 타이밍 벨트 |
| GR = 3:1일 때 출력 토크 | τ_out = τ_in × 3 |
