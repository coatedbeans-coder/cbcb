# 캠과 팔로워 (Cam & Follower) (★★★)

#cam #mechanism

## Overview Table (한눈에 비교)

| 항목 | 설명 |
|------|------|
| **캠 (Cam)** | 회전 운동 → 직선·진동 운동으로 변환하는 비원형 디스크 |
| **팔로워 (Follower)** | 캠 윤곽을 따라 움직이는 부품 |
| **스트로크 / Throw** | 팔로워가 이동하는 최대 거리 (최대 반지름 − 최소 반지름) |
| **드웰 (Dwell)** | 일정 반지름 구간 → 팔로워 정지 |
| **로브 (Lobe)** | 캠의 돌출 부분 = 팔로워를 밀어 올리는 부분 |
| **이벤트 (Event)** | 1회전 내 로브 1개가 만드는 운동 사이클 |

## 캠의 종류

### Edge Cam (엣지 캠 / 디스크 캠)
가장 기본적인 형태. 팔로워가 캠 가장자리를 따라 움직인다.

```
           ┌──── 로브(최고점) ────┐
     팔로워 │       ↑ throw        │
      │     │  ┌──────────┐       │
      ↓     │  │  캠 디스크 │       │
     ══════ └──┘          └───────┘
              ← 드웰(일정 반지름) →
```

- **편심 캠 (Eccentric Cam)**: 원형이지만 회전 중심이 오프셋 → 단순 왕복 운동
- **로브형 캠**: 돌출부로 팔로워 밀어 올림 → 복잡한 운동 패턴 가능
- **스네일 캠 (Snail Cam)**: 한 방향 구동만 허용. 팔로워가 급격히 복귀.
- **래칫형 캠**: 로브 여러 개로 1회전 중 복수 이벤트 생성

> [!tip] 팔로워 접촉점 형태
> - **평면(Flat)**: 단순, 제작 쉬움. 고속 시 마모 큼
> - **롤러(Roller)**: 마찰 감소 ★ 권장. 베어링 장착 팔로워
> - **나이프 엣지**: 정밀 추종, 하중에 약함

### Face Cam (페이스 캠 / 판 캠)
캠 면에 홈을 파서 팔로워 핀이 홈을 따라 이동.

- **장점**: 양방향 강제 구동(positive-drive) → 팔로워가 홈에 잠겨 있어 스프링 불필요
- **단점**: 제작 어려움, 홈 가공 필요

### Drum Cam (드럼 캠 / 배럴 캠)
원기둥 외부에 홈. 팔로워가 홈을 따라 캠 축 방향으로 왕복.

```
  ←→ 팔로워 왕복 (축 방향)
  
  ┌────────────────────┐
  │ ~~~홈의 나선경로~~~  │ 회전
  └────────────────────┘
```

## 말 갤럽용 캠 설계 전략

> [!important] 핵심 설계
> 말의 다리 4개를 하나의 캠샤프트에 90° 위상 간격으로 배치:
>
> ```
> 캠샤프트 (1개 모터, 갤럽 순서: 우뒤→좌뒤→우앞→좌앞)
>   ├── 캠A (0°)   → 우뒤다리
>   ├── 캠B (90°)  → 좌뒤다리
>   ├── 캠C (180°) → 우앞다리
>   └── 캠D (270°) → 좌앞다리
> ```
>
> 각 캠의 1회전 = 갤럽 1주기

**각 캠 프로파일 (1다리 기준):**
```
  0°~90°:   상승 (다리 들기)      [리프트 90°]
  90°~120°: 드웰 (공중 유지)     [공중 드웰 30°]
  120°~270°: 하강 (착지)         [착지 150°]
  270°~360°: 드웰 (착지 유지)    [착지 드웰 90°]
  ※ 말 갤럽 메커니즘 설계.md와 동일한 프로파일
```

**캠 크기 권고 (총 높이 60cm 조형물 기준):**
- 스트로크(다리 움직임) = 15~25mm 목표 시
- 캠 직경 최소 50~70mm (스트로크의 2~3배)

## 제작 팁

| 상황 | 권장 재료 |
|------|-----------|
| 프로토타입 | 6mm 합판 CNC 절삭 or 레이저 컷 |
| 최종 제품 | 알루미늄 또는 아크릴 |
| 팔로워 접촉 | 볼 베어링 롤러 달기 → 마찰 최소화 |

> [!warning] 캠 크기 함정
> 캠이 너무 작으면 팔로워를 충분히 밀지 못함 (force leverage 부족)
> → 크게 설계해도 공간 허용 범위 내에서 여유 있게

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "회전 → 정지 포함 왕복 운동" | 드웰 구간이 있는 캠 |
| "양방향 강제 구동" | 페이스 캠 (홈형) |
| "1회전에 4번 왕복" | 로브 4개짜리 캠 |
| "스트로크 = ?" | 최대 반지름 − 최소 반지름 |
| "팔로워 마찰 최소화" | 롤러 팔로워 |
