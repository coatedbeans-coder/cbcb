## 노이즈 필터링

### 하드웨어 필터 (RC 로우패스)

```
         R (10kΩ)
  Vin ──┤├──────── Vout (Arduino A0)
                │
               [C] (100nF)
                │
               GND

차단 주파수: fc = 1 / (2π × R × C)
fc = 1 / (2π × 10000 × 0.0000001) ≈ 159 Hz

159Hz 이하만 통과 → 모터 고주파 노이즈 차단
```

### 소프트웨어 이동 평균 필터

```cpp
const int WINDOW = 10;
int buf[WINDOW];
int idx = 0;
long total = 0;

int smoothRead(int pin) {
  total -= buf[idx];
  buf[idx] = analogRead(pin);
  total += buf[idx];
  idx = (idx + 1) % WINDOW;
  return total / WINDOW;
}
```

## 전압 레벨 변환 (3.3V ↔ 5V)

MPU-6050, BME280 등 3.3V 센서 → Arduino 5V I2C에 직접 연결 금지.

```
간단한 레벨 시프터 (1방향):
  5V 신호 → R1 (1kΩ) → R2 (2kΩ) → GND
  R1-R2 접점 → 3.3V 신호
  (전압 분배기: 5V × 2/(1+2) = 3.33V)
  ⚠️ 단방향만 가능 — I2C처럼 양방향 통신에는 사용 불가

양방향 레벨 시프터 모듈:
  TXS0108E 또는 MOSFET 기반 모듈 구매 권장
  I2C에는 양방향 레벨 시프터 필수
```

## 풀업/풀다운 저항

```
풀업 저항 (Pull-up):
  +5V ─── R(10kΩ) ─── 핀 ─── 버튼 ─── GND
  버튼 열림: 핀 = HIGH (5V)
  버튼 닫힘: 핀 = LOW (GND)

내부 풀업 사용 (간편):
  pinMode(pin, INPUT_PULLUP);
  → 내부 20~50kΩ 풀업 자동 활성화
```
