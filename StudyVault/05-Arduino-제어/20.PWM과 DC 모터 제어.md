# PWM과 DC 모터 제어 (★★)

#arduino #motor-dc #control

## Overview Table

| 항목 | 설명 |
|------|------|
| **PWM** | 펄스 폭 변조 — 디지털로 아날로그 전압 흉내 |
| **analogWrite(pin, value)** | 0~255, PWM 핀에만 사용 |
| **H-Bridge** | DC 모터 방향 + 속도 제어 |
| **기어드 DC 모터** | 캠샤프트 연속 구동에 최적 |

## PWM 원리

```
value=255: 항상 HIGH ──────────────── 5V
value=128: 50% 듀티  ████░░░░████░░░  약 2.5V
value=64:  25% 듀티  ██░░░░░░██░░░░░  약 1.25V
value=0:   항상 LOW  ░░░░░░░░░░░░░░░  0V

주파수: 핀 3,11 → 490Hz / 핀 5,6 → 980Hz (Uno 기본)
```

## 기본 DC 모터 속도 제어

```cpp
// L298N 기준
int ENA = 9;  // PWM 핀 (속도)
int IN1 = 7;  // 방향 1
int IN2 = 8;  // 방향 2

void setup() {
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
}

// 정방향, speed: 0~255
void forward(int speed) {
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, speed);
}

// 역방향
void backward(int speed) {
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, HIGH);
  analogWrite(ENA, speed);
}

// 정지
void stopMotor() {
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, 0);
}
```

## 캠샤프트 속도 제어 (포텐시오미터)

```cpp
void loop() {
  int pot = analogRead(A0);                    // 0~1023
  int speed = map(pot, 0, 1023, 50, 255);     // 최소 50 → 탈조 방지
  forward(speed);
  
  // (실제 갤럽 Hz는 기어비·캠 설계에 따라 달라짐 — 참고용 근사값 아님)
  // 표시 (시리얼 모니터)
  Serial.print("Speed: ");
  Serial.println(speed);
}
```

## 소프트 스타트 (부드러운 가속)

```cpp
// ⚠️ 단독 동작 전용 — 다른 장치와 동시 구동 시 millis() 기반으로 재작성 필요
void softStart(int targetSpeed, int stepDelay) {
  for (int s = 0; s <= targetSpeed; s += 5) {
    forward(s);
    delay(stepDelay);
  }
}

void softStop(int currentSpeed, int stepDelay) {
  for (int s = currentSpeed; s >= 0; s -= 5) {
    forward(s);
    delay(stepDelay);
  }
  stopMotor();
}
```

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "50% 속도 PWM 값" | `analogWrite(pin, 128)` |
| "PWM Uno 주파수 (핀5)" | 980Hz |
| "DC 모터 역방향" | IN1=LOW, IN2=HIGH |
| "부드러운 가속" | 소프트 스타트 (값 점진 증가) |
