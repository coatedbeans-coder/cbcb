# I2C와 SPI 통신 (★★)

#arduino #control

## Overview Table

| 프로토콜 | 선 수 | 속도 | 장치 수 | 용도 |
|----------|-------|------|---------|------|
| **I2C** | 2선 (SDA, SCL) | 100kHz~400kHz | 최대 127개 | 서보 드라이버, 센서 |
| **SPI** | 4선 (MOSI, MISO, SCK, SS) | 수 MHz | SS 핀 수만큼 | 고속 통신 |
| UART/Serial | 2선 (TX, RX) | 9600~115200bps | 1:1 | PC 통신, GPS |

## I2C 개요

```
마스터 (Arduino)                슬레이브1      슬레이브2
     │──SDA──────────────────┤ 0x40 ├──────┤ 0x68 ├──
     │──SCL──────────────────┤      ├──────┤      ├──
     │                       └──────┘      └──────┘
     GND ─────────────────────────────────────────────
     
  SDA: 데이터 (A4 on Uno)
  SCL: 클럭  (A5 on Uno)
  풀업 저항 4.7kΩ 필요 (대부분 모듈에 내장)
```

**I2C 주소:** 각 장치마다 고유 7비트 주소 (0x08~0x77)

## Wire 라이브러리 (I2C)

```cpp
#include <Wire.h>

void setup() {
  Wire.begin();           // 마스터로 초기화
  Serial.begin(9600);
}

// 데이터 쓰기
void writeRegister(uint8_t addr, uint8_t reg, uint8_t value) {
  Wire.beginTransmission(addr); // 장치 주소 선택
  Wire.write(reg);              // 레지스터 주소
  Wire.write(value);            // 데이터
  Wire.endTransmission();
}

// 데이터 읽기
uint8_t readRegister(uint8_t addr, uint8_t reg) {
  Wire.beginTransmission(addr);
  Wire.write(reg);
  Wire.endTransmission(false); // repeated start
  Wire.requestFrom(addr, 1);   // 1바이트 요청
  return Wire.read();
}
```

## PCA9685 서보 드라이버 (I2C, 16채널)

> [!important] 프로젝트 핵심 모듈
> 서보 5개 이상 제어 시 Arduino 핀 부족 → PCA9685로 해결
>
> - I2C 주소: 0x40 (기본), 주소 핀으로 0x40~0x7F 설정 가능
> - 16채널 PWM, 각 12비트 분해능
> - 서보 전원 별도 연결 (V+에 6V 공급)

```cpp
#include <Wire.h>
#include <Adafruit_PWMServoDriver.h>

// 기본 주소 0x40, 다른 주소: Adafruit_PWMServoDriver(0x41)
Adafruit_PWMServoDriver pwm = Adafruit_PWMServoDriver();

#define SERVOMIN  150   // 0°에 해당하는 카운트값
#define SERVOMAX  600   // 180°에 해당하는 카운트값
#define SERVO_FREQ 50   // 서보 주파수 50Hz

void setup() {
  pwm.begin();
  pwm.setOscillatorFrequency(27000000); // 정밀도 보정
  pwm.setPWMFreq(SERVO_FREQ);
  delay(10);
}

void setServoAngle(uint8_t channel, uint8_t angle) {
  uint16_t pulselen = map(angle, 0, 180, SERVOMIN, SERVOMAX);
  pwm.setPWM(channel, 0, pulselen);
}

// ⚠️ 정적 포지션 데모용 — 실제 갤럽 구현 시 millis() 기반 논블로킹 방식 사용
void loop() {
  // 다리 4개 제어 (채널 0~3)
  setServoAngle(0, 45);  // 우앞 다리
  setServoAngle(1, 0);   // 우뒤 다리
  setServoAngle(2, 45);  // 좌앞 다리
  setServoAngle(3, 0);   // 좌뒤 다리
  delay(500);
}
```

## SPI 개요

```
마스터 (Arduino)    슬레이브
   MOSI(11) ──────→ MOSI
   MISO(12) ←────── MISO
   SCK (13) ──────→ SCK
   SS  (10) ──────→ CS (Chip Select)
```

```cpp
#include <SPI.h>

void setup() {
  SPI.begin();
  pinMode(10, OUTPUT); // SS 핀
  digitalWrite(10, HIGH); // 비활성화 상태
}

void writeData(byte data) {
  digitalWrite(10, LOW);  // 장치 선택
  SPI.transfer(data);     // 데이터 전송
  digitalWrite(10, HIGH); // 해제
}
```

## I2C 스캐너 (주소 확인)

새 I2C 모듈 연결 시 주소 확인에 사용.

```cpp
#include <Wire.h>

void setup() {
  Wire.begin();
  Serial.begin(9600);
  
  for (byte addr = 1; addr < 127; addr++) {
    Wire.beginTransmission(addr);
    if (Wire.endTransmission() == 0) {
      Serial.print("I2C 장치 발견: 0x");
      Serial.println(addr, HEX);
    }
  }
}

void loop() {}
```

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "I2C 최소 배선" | 신호선 2선 (SDA, SCL) + GND 공통 + 전원 → 실질 4선 |

| "16채널 서보 드라이버" | PCA9685 (I2C) |

| "I2C 주소 모를 때" | I2C 스캐너 코드 실행 |
