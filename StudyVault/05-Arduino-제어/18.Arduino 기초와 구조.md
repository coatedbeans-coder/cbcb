## 스케치 기본 구조

```cpp
// 전처리 & 변수 선언
#include <Servo.h>          // 라이브러리 포함
const int LED_PIN = 13;     // 상수 정의
int sensorValue = 0;        // 변수 선언

void setup() {
  // 1번만 실행 — 초기화
  pinMode(LED_PIN, OUTPUT); // 핀 모드 설정
  Serial.begin(9600);       // 시리얼 통신 시작
}

void loop() {
  // 무한 반복
  digitalWrite(LED_PIN, HIGH); // 핀 HIGH
  delay(1000);                 // 1초 대기
  digitalWrite(LED_PIN, LOW);  // 핀 LOW
  delay(1000);
}
```

## 핵심 함수 레퍼런스

### 디지털 I/O

```cpp
pinMode(pin, INPUT/OUTPUT/INPUT_PULLUP);
digitalWrite(pin, HIGH/LOW);
int val = digitalRead(pin);   // 반환: HIGH(1) or LOW(0)
```

### 아날로그

```cpp
int val = analogRead(A0);     // 0~1023 (10비트)
analogWrite(pin, value);       // 0~255 PWM (PWM 핀만)
```

> [!warning] `analogWrite()`는 PWM 핀만!
> Uno PWM 핀: 3, 5, 6, 9, 10, 11
> 서보에는 `analogWrite()` 사용 금지 → `Servo.write()` 필수

### 시간

```cpp
delay(ms);           // ms 밀리초 대기 (blocking!)
delayMicroseconds(us); // us 마이크로초 대기
unsigned long t = millis(); // 시작부터 경과 ms
unsigned long t = micros(); // 시작부터 경과 us
```

## 시리얼 통신 (디버깅)

```cpp
Serial.begin(9600);           // setup()에서 초기화
Serial.print("Value: ");      // 줄바꿈 없이 출력
Serial.println(sensorValue);  // 줄바꿈 포함 출력
Serial.print(val, DEC);       // 10진수
Serial.print(val, HEX);       // 16진수
```

## 변수 타입

| 타입 | 크기 | 범위 |
|------|------|------|
| `int` | 2 bytes | -32,768 ~ 32,767 |
| `unsigned int` | 2 bytes | 0 ~ 65,535 |
| `long` | 4 bytes | -2,147,483,648 ~ 2,147,483,647 |
| `unsigned long` | 4 bytes | 0 ~ 4,294,967,295 |
| `float` | 4 bytes | ±3.4×10^38 (소수점) |
| `byte` | 1 byte | 0 ~ 255 |
| `bool` | 1 byte | true/false |
| `char` | 1 byte | -128 ~ 127 |

> [!important] millis() 오버플로우
> `millis()`는 `unsigned long` 필수!
> 약 49.7일 후 0으로 리셋됨.
> 비교 시: `if (millis() - lastTime >= interval)` 방식으로 안전하게.

## 내부 풀업 저항

```cpp
pinMode(buttonPin, INPUT_PULLUP); // 내부 풀업 활성화

// 버튼 누름 = LOW (반전 로직!)
if (digitalRead(buttonPin) == LOW) {
  // 버튼 눌림
}
```

## map() 함수

값 범위 변환에 유용.

```cpp
// map(value, fromLow, fromHigh, toLow, toHigh)
int angle = map(sensorValue, 0, 1023, 0, 180);
// 아날로그 입력(0~1023) → 서보 각도(0~180)
```
