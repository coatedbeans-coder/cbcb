## 5월

**1주** 기획 및 기초 설계 — 말 보행 메커니즘 분석, 갤럽 운동 레퍼런스 조사, 작품 전체 구조 스케치, 크기·무게·중심 초안 설정, 구동 방식 결정

**2주** 세부 설계 ① — 말 구동부 설계 (관절 수, 모터 배치, 프레임 재질 결정)

**3주** 세부 설계 ② — 인형 구조 설계 (구체관절 위치, 크기 비례, 앞으로 숙인 탑승 자세 고정 방식, 활 겨누는 팔 자세 설계, 곡궁 설계 포함)

**4주** 블렌더 시뮬레이션 — 말 보행 동작 시뮬레이션, 무게중심 검토, 설계 문제점 도출

**5주** 재료 리스트업 및 공수 시작 — 시뮬레이션 결과 반영 설계 수정, 필요 부품·재료 전체 목록 작성, 발주 시작

  

## 6월

**1주** 재료·장비 공수 완료 및 검수 — 입고 확인, 부족 재료 추가 발주, 작업 환경 세팅

**2주** 말 프레임 제작 ① — 몸체 골격(프레임) 제작, 하중 테스트

**3주** 말 프레임 제작 ② — 다리 관절 기구부 조립, 수동 동작 확인

**4주** 구동부 제작 ① — 모터 마운트 설계 및 제작, 단일 관절 구동 테스트

  

## 7월

**1주** 구동부 제작 ② — 전체 다리 연동 구동 테스트, 타이밍 조정

**2주** 구동부 제작 ③ — 제어 회로 구성, 펌웨어 기초 작성, 동작 패턴 입력

**3주** 구동부 단독 테스트 및 수정 — 갤럽 동작 반복 테스트, 진동·소음·발열 점검, 문제점 수정

**4주** 말 외형 제작 ① — 프레임 위 폼/점토 등으로 말 외형 조형 시작

  

## 8월

**1주** 말 외형 제작 ② — 외형 조형 완성, 표면 처리 (샌딩, 프라이머)

**2주** 말 외형 마감 — 도색, 질감 표현 (털 느낌, 눈·코·입 디테일), 장식 마구 제작

**3주** 말 전체 통합 테스트 — 외형+구동부 결합 후 동작 테스트, 외형 손상 여부 확인, 수정

**4주** 말 후가공 및 완성 — 수정 사항 반영, 마감재 도포, 말 파트 완성 선언

  

## 9월

**1주** 인형 제작 ① — 구체관절 헤드 조형 (석분점토 또는 폴리머 클레이), 얼굴 디테일 조각

**2주** 인형 제작 ② — 몸통·팔·다리 구체 제작, 관절 소켓 가공

**3주** 인형 제작 ③ — 전체 관절 조립, 앞으로 숙인 탑승 자세 고정 테스트, 활 겨누는 팔 동작 범위 확인

**4주** 인형 후가공 — 표면 샌딩·도색, 눈 제작(유리눈 또는 아크릴 페인팅), 가발 제작 또는 구매·가공

  

## 10월

**1주** 인형+말 결합 테스트 — 인형 탑승 고정, 무게중심 재확인, 앞으로 숙인 자세에서 구동 시 인형 흔들림 점검

**2주** 의상 디자인 확정 — 말 치수·인형 치수 기반 의상 패턴 설계, 비즈자수 배치 디자인, 원단 선정 및 추가 재료 발주

**3주** 의상 제작 ① — 기본 원단 재단·봉제 (상의 베이스, 스커트 베이스)

**4주** 의상 제작 ② — 비즈자수 작업 시작 (상의 자수 집중)

  

## 11월

**1주** 의상 제작 ③ — 비즈자수 작업 계속 (스커트·소매 자수)

**2주** 의상 제작 ④ — 의상 마감 봉제, 전체 자수 완성, 인형에 착장 테스트

**3주** 소품 제작 — 두파타, 장신구, 마구 장식 등 소품 완성, 곡궁 제작 및 인형 손 고정

**4주** 분장 및 착장 완성 — 인형 메이크업 페인팅, 의상·소품 최종 착장, 전체 비주얼 확인

  

## 12월

**1주** 최종 통합 테스트 ① — 인형+의상+말+구동부 완전 결합 상태로 동작 테스트, 의상 걸림·손상 여부 확인

**2주** 최종 통합 테스트 ② 및 튜닝 — 동작 속도·타이밍 미세 조정, 소음 저감, 진동 완충 처리

**3주** 문서 정리 및 평가 — 제작 과정 기록 정리, 설계도 최종본 저장, 잘된 점·아쉬운 점 평가 작성

**4주** 재료 정리 + 향후 계획 수립 + 휴식 — 잔여 재료 정리·보관, 다음 프로젝트 아이디어 메모, 충분한 휴식
