## 🔢 핵심 공식 → [[7.힘과 토크]]

| 공식 | 설명 | 단위 |
|------|------|------|
| `τ = F × r` | 토크 = 힘 × 반지름 | N·m |
| `P = τ × ω` | 파워 = 토크 × 각속도 | W |
| `ω = 2π × n / 60` | 각속도 (rad/s) ← RPM 변환 | rad/s |
| `F = m × a` | 뉴턴 2법칙 | N |
| `W = m × g` | 무게 (g = 9.81 m/s²) | N |
| `ME = F_out / F_in` | 기계적 이득 | 무차원 |
| `μ = F_friction / N` | 마찰계수 = 마찰력 / 수직력 | 무차원 |

## ⚙️ 기어비 & 속도 → [[5.기어와 동력전달]]

| 공식 | 설명 |
|------|------|
| `GR = N_driven / N_driver` | 기어비 = 피동 이 수 / 구동 이 수 |
| `ω_out = ω_in / GR` | 출력 회전속도 |
| `τ_out = τ_in × GR` | 출력 토크 (이상적) |
| `벨트 속도비 = d_driver / d_driven` | 풀리 지름 비 |

## 🐎 말 갤럽 타이밍 레퍼런스 → [[말 갤럽 메커니즘 설계]]

```
갤럽 보행 순서 (4박자):
  [우뒤] → [좌뒤] → [우앞] → [좌앞]
  각 다리 위상 오프셋: 0° / 90° / 180° / 270°

캠 로브 설계:
  - 이벤트(1 다리 올림): 90° 회전
  - 드웰(착지): 30° 회전
  - 캠샤프트 1회전 = 갤럽 1주기
```

## 🎛️ 서보 제어 → [[19.서보 제어 프로그래밍]]

```cpp
#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(9);   // 핀 9에 연결
}

// ⚠️ 단일 서보 테스트 전용 — 실제 작품에는 millis() 기반 논블로킹 방식 사용
void loop() {
  myServo.write(0);    // 0도 (최소)
  delay(1000);
  myServo.write(90);   // 90도 (중앙)
  delay(1000);
  myServo.write(180);  // 180도 (최대)
  delay(1000);
}
```

> [!warning] 주의
> `analogWrite()`로 서보 제어 **불가** — 반드시 `Servo` 라이브러리 사용

## 🔄 스테퍼 모터 제어 → [[16.스테퍼 모터]]

```cpp
#include <Stepper.h>

const int STEPS = 200;          // 1회전당 스텝 수
Stepper stepper(STEPS, 2, 3, 4, 5);

void setup() {
  stepper.setSpeed(30);         // RPM 설정
}

void loop() {
  stepper.step(200);            // 1바퀴 정방향
  stepper.step(-200);           // 1바퀴 역방향
}
```

// L298N H-Bridge 기준

## 📐 캠 설계 요점 → [[3.캠과 팔로워]]

| 항목 | 설명 |
|------|------|
| **Throw (스트로크)** | 최고점 반지름 - 최저점 반지름 = 팔로워 이동 거리 |
| **Dwell** | 일정 반지름 구간 → 팔로워 정지 |
| **이벤트 수** | 1회전당 로브 개수 |
| **Edge cam** | 회전 → 직선 운동 (가장 간단) |
| **Face cam** | 양방향 구동 가능 (홈 방식) |

## 🔗 4절 링크 — Grashof 법칙 → [[4.4절 링크장치]]

```
링크 길이: s(최단) + l(최장) < p + q (나머지 둘)
→ 조건 만족 시: 최단 링크가 크랭크(완전 회전) 가능
→ 조건 불만족: 모든 링크가 로커(왕복)만 가능
```

## 📏 단위 변환 참조

| 변환 | 공식 |
|------|------|
| RPM → rad/s | `× π / 30` |
| rad/s → RPM | `× 30 / π` |
| N·m → kgf·cm | `× 10.197` |
| kgf·cm → N·m | `× 0.0981` |
| mm → in | `÷ 25.4` |
| in → mm | `× 25.4` |
