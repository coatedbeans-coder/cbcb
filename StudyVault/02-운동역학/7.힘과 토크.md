# 힘과 토크 (Force & Torque) (★★★)

#dynamics #torque

## Overview Table

| 물리량 | 기호 | 단위 | 설명 |
|--------|------|------|------|
| 힘 (Force) | F | N (뉴턴) | 질량 × 가속도 |
| 무게 (Weight) | W | N | mg (g = 9.81 m/s²) |
| 토크 (Torque) | τ | N·m | 회전력 = 힘 × 반지름 |
| 파워 (Power) | P | W (와트) | 토크 × 각속도 |
| 각속도 | ω | rad/s | 2π × RPM / 60 |
| 기계적 이득 | MA | 무차원 | F_out / F_in |

---

## 핵심 공식

### 뉴턴의 법칙
```
F = m × a
  F: 힘 (N)
  m: 질량 (kg)
  a: 가속도 (m/s²)

무게: W = m × g  (g = 9.81 m/s²)
```

### 토크
```
τ = F × r
  τ: 토크 (N·m)
  F: 힘 (N)
  r: 힘 팔 / 반지름 (m)

또는 kgf·cm 단위: τ[N·m] = τ[kgf·cm] × 0.0981
```

### 파워 & 속도
```
P = τ × ω = F × v
  P: 파워 (W)
  ω: 각속도 (rad/s)
  v: 선속도 (m/s)

ω = 2π × n / 60   (n: RPM)
v = ω × r
```

## 프로젝트 토크 계산 (말 다리)

> [!important] 모터 사이징 계산 예시
>
> **가정 (총 높이 30cm 기준):**
> - 말 다리 조립체 무게: 120g = 0.12 kg
> - 무게 중심까지 거리 (팔 길이): 33mm = 0.033 m
> - 동작 가속도 고려 안전계수: 2배
>
> **필요 토크:**
> ```
> F = W = 0.12 × 9.81 = 1.177 N
> τ_필요 = F × r = 1.177 × 0.033 = 0.039 N·m
>        = 0.039 / 0.0981 ≈ 0.4 kgf·cm
>
> 안전계수 적용: 0.4 × 2 = 0.8 kgf·cm
>
> → 서보: 2 kgf·cm 이상 선택 → MG90S (2.2 kgf·cm) 가능 / MG996R (13 kgf·cm) 여유 충분
> ```

## 자유 물체도 (Free Body Diagram)

복잡한 메커니즘에서 하중 분석을 위한 도구.

```
단계:
1. 분석할 부품 하나 선택 (다리, 링크 등)
2. 모든 외부 힘을 화살표로 표시
   - 중력 (아래 방향)
   - 반력 (지지점)
   - 모터 힘
   - 마찰력
3. ΣF = 0 (정적), ΣF = ma (동적) 적용
4. 미지 힘 계산
```

## 기계적 이득 (Mechanical Advantage)

```
MA = F_출력 / F_입력 = r_입력 / r_출력

MA > 1 : 힘 증폭 (토크는 감소)
MA < 1 : 속도 증폭 (힘은 감소)
MA = 1 : 힘·속도 동일 (방향 전환)
```

> [!tip] 이상적 vs 실제
> 실제 MA는 마찰 손실로 이론값보다 항상 낮음.
> 효율(η) = 실제 MA / 이론 MA
> 일반 기어: η ≈ 90~95%, 웜 기어: η ≈ 40~90%

## 토크와 모터 선택

| 모터 종류 | 토크 스펙 단위 | 변환 |
|-----------|---------------|------|
| 서보 모터 | kgf·cm | × 0.0981 = N·m |
| DC 모터 | mN·m | ÷ 1000 = N·m |
| 스테퍼 모터 | N·cm | ÷ 100 = N·m |

**선택 기준:**
```
실제 필요 토크 ≤ 모터 rated torque × 0.7
(정격의 70% 이하로 사용 = 30% 여유)
```

> [!tip] 안전계수 적용 방법 두 가지 — 모두 유효
> **방법 A (계산 기반)**: 필요 토크 × 2~3 ≤ 서보 스펙
>   → 동적·정적 토크 합산 후 2~3배 여유 확보 (전시용 장시간 구동 **권장**)
> **방법 B (정격 기반)**: 필요 토크 ≤ 서보 스펙 × 0.7
>   → "정격의 70% 이하로 운용" (≈ 1.43배 최소 여유)
> 두 기준은 목적이 다르며 동시에 만족하는 서보를 선택하면 가장 안전.

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "토크 = ?" | `τ = F × r` |
| "파워 = ?" | `P = τ × ω` |
| "RPM → rad/s" | `ω = 2π × RPM / 60` |
| "500g 물체, 10cm 팔, 필요 토크" | `W = 4.9N, τ = 0.49 N·m ≈ 5 kgf·cm` |
| "MA > 1이면" | 힘 증폭, 속도 감소 |
