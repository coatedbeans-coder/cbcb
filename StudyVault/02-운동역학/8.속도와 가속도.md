# 속도와 가속도 (Velocity & Acceleration) (★★)

#dynamics

## Overview Table

| 물리량 | 기호 | 단위 | 설명 |
|--------|------|------|------|
| 변위 (Displacement) | x | m | 위치 변화량 (방향 있음) |
| 속력 (Speed) | v | m/s | 단위 시간당 이동 거리 (방향 없음) |
| 속도 (Velocity) | v | m/s | 방향이 있는 속력 |
| 가속도 (Acceleration) | a | m/s² | 단위 시간당 속도 변화율 |

---

## x, v, a의 관계

```
  x ←── 적분 ──── v ←── 적분 ──── a
  x ───→ 미분 ──→ v ───→ 미분 ──→ a

  v = dx/dt    (위치의 미분 = 속도)
  a = dv/dt    (속도의 미분 = 가속도)

그래프 관계:
  x-t 그래프 기울기 = 속도
  v-t 그래프 기울기 = 가속도
  v-t 그래프 면적 = 변위
```

## 등가속도 운동 공식

> [!important] 4가지 기본 공식
>
> ```
> v = v₀ + a·t
> x = v₀·t + ½·a·t²
> v² = v₀² + 2·a·x
> x = (v₀ + v)/2 · t
> ```
>
> - v₀: 초기 속도
> - v: 최종 속도
> - a: 가속도
> - t: 시간
> - x: 이동 거리

## 말 갤럽 운동에서의 적용

### 다리 올림 단계 (가속 구간)

```
가정 (30cm 조형물 기준, 스트로크 ~10mm, 들기 시간 200ms):
  다리가 정지 상태(v₀=0)에서 10mm 올라가는데 0.2초
  
구하기: 평균 가속도

  x = ½ · a · t²  (v₀=0이므로)
  0.01 = ½ · a · (0.2)²
  a = 0.01 / (0.5 × 0.04) = 0.5 m/s²

최대 속도(중간 지점): v = a·t = 0.5 × 0.1 = 0.05 m/s = 50mm/s
```

### 갤럽 주기 계산

```
갤럽 주기 = 1초 (1 Hz) → 느린 갤럽 (캔터 수준)
서보 1주기 = 1000ms

갤럽 주기 = 0.5초 (2 Hz) → 빠른 갤럽
서보 1주기 = 500ms
```

## 무대기계 Hendrickson의 가속도 프로파일

> [!tip] 사다리꼴 속도 프로파일 (Trapezoidal)
> 무대 기계에서 가장 많이 사용하는 이동 프로파일.
>
> ```
> 속도
>  ↑   ┌──────────────┐
>  │  /│              │\
>  │ / │              │ \
>  │/  │              │  \
>  └───┴──────────────┴───→ 시간
>   가속  등속 구간   감속
> ```
>
> - 가속 구간: 최대 힘 필요 (모터 선택 기준)
> - 등속 구간: 마찰만 극복
> - 감속 구간: 제동력 필요

## 각운동 vs 직선운동

| 직선 | 각운동 | 관계 |
|------|--------|------|
| x (m) | θ (rad) | x = θ × r |
| v (m/s) | ω (rad/s) | v = ω × r |
| a (m/s²) | α (rad/s²) | a = α × r |
| F (N) | τ (N·m) | τ = F × r |
| m (kg) | I (kg·m²) | — |

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "등가속, v₀=0, 시간·거리 주어짐" | `a = 2x / t²` (예: x=10mm, t=0.2s → 0.5 m/s²) |
| "v-t 그래프의 면적" | 이동 거리 (변위) |
| "사다리꼴 프로파일 가속구간" | 최대 힘 필요, 모터 선택 기준 |
| "v = ω × r 에서 r이 클수록" | 선속도(v) 커짐 |
