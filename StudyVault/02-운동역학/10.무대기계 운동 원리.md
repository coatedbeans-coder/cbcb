# 무대기계 운동 원리 (★★)

#dynamics

> [!tip] 이 책의 관점
> Hendrickson의 **무대기계 설계**는 "큰 하중을 안전하게, 부드럽게 움직이는" 것에 집중.
> 메카트로닉스 아트에 직접 응용 가능한 설계 철학을 담고 있음.

---

## 운동 프로파일 (Motion Profile)

### 사다리꼴 프로파일 (가장 일반적)

```
속도 (v)
│        ┌──────────────┐
│      ╱ │   등속 구간    │ ╲
│    ╱   │              │   ╲
│  ╱     │              │     ╲
│╱       │              │       ╲
└────────────────────────────────→ 시간 (t)
  가속    │              │   감속
  구간    └──────────────┘   구간
```

**세 구간:**
1. **가속 구간**: 정지 → 최대 속도. 최대 힘(토크) 필요
2. **등속 구간**: 최대 속도 유지. 마찰만 극복하면 됨
3. **감속 구간**: 최대 속도 → 정지. 제동력(braking) 필요

### S-커브 프로파일 (고급)

```
속도 (v)
│         ╭────────────╮
│       ╭╯              ╰╮
│     ╭╯                  ╰╮
│   ╭╯                      ╰╮
└────────────────────────────→ 시간 (t)
```

- 가속/감속이 부드러움 (저크 최소화)
- 기계 수명 연장, 소음 감소
- Arduino에서 millis() + 수식으로 구현 가능

## 가속 구간에서의 힘 계산

> [!important] 기동 토크 계산
>
> ```
> F_total = F_inertia + F_friction + F_gravity
>
> F_inertia = m × a  (관성력)
> F_friction = μ × N  (마찰력)
> F_gravity = m × g × sin(θ)  (경사 시)
>
> 수평 운동이고 마찰만 있을 때:
> F_total = m × a + μ × m × g
>         = m × (a + μ × g)
> ```

## 무대기계 설계 철학 → 아트 프로젝트 적용

| 무대기계 원칙 | 아트 프로젝트 적용 |
|---------------|-------------------|
| 안전계수 최소 5배 | 최소 2~3배 (전시 환경) |
| 이중 안전 장치 | 리미트 스위치 추가 |
| 부드러운 시작/정지 | S-커브 또는 사다리꼴 프로파일 |
| 점검 및 유지보수 | 베어링 접근 가능 설계 |
| 하중 경로 명확히 | 프레임 하중 분산 설계 |

## 말 갤럽 운동 프로파일 설계

```
갤럽 1주기 (1초 가정):

  ┌──────────────────────────────────────────────────┐
  │ 다리 위상도 (갤럽 순서: 우뒤→좌뒤→우앞→좌앞)              │
  │                                                  │
  │ 우뒤: ██░░░░░░██░░░░░░  (0% 시작, 들림=██)          │
  │ 좌뒤: ░░██░░░░░░██░░░░  (25% 오프셋)                │
  │ 우앞: ░░░░██░░░░░░██░░  (50% 오프셋)                │
  │ 좌앞: ░░░░░░██░░░░░░██  (75% 오프셋)                │
  └──────────────────────────────────────────────────┘

  각 다리: 25% 공중 (250ms), 75% 착지 (750ms)
  캠 로브 비율: 90° 상승 + 30° 드웰(공중) + 150° 하강 + 90° 드웰(착지)
```

## 변위-속도-가속도 그래프 읽기

```
x(t) → 사인파 (캠 회전 시)
v(t) → 코사인파 (x의 미분)
a(t) → -사인파 (v의 미분)

  가속도 최대 위치 = 속도 = 0 (방향 전환)
  속도 최대 위치 = 가속도 = 0 (중간 지점)
```

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "최대 모터 토크 필요 시점" | 가속 구간 시작 시 |
| "등속 구간에서 필요한 힘" | 마찰력만 극복 |
| "기계 충격 최소화 방법" | S-커브 속도 프로파일 |
| "관성력 = ?" | `F = m × a` |
| "사다리꼴 프로파일의 3구간" | 가속 / 등속 / 감속 |
