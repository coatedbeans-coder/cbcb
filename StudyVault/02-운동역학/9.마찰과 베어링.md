## 마찰의 기본

```
F_마찰 = μ × N

  F_마찰: 마찰력 (N)
  μ: 마찰계수 (무차원)
  N: 수직 하중 (N)
```

**정지 마찰 vs 운동 마찰:**
```
μ_정지 > μ_운동
→ 멈춰있는 물체를 움직이기 시작할 때 더 큰 힘 필요
→ 기동 토크 > 유지 토크
```

---

## 재료별 마찰계수 (대략값)

| 재료 조합 | μ (건조) | μ (윤활) |
|-----------|----------|----------|
| 금속 on 금속 | 0.15~0.3 | 0.05~0.1 |
| 목재 on 목재 | 0.25~0.5 | — |
| 플라스틱 on 금속 | 0.1~0.3 | 0.05~0.1 |
| 테플론 on 금속 | 0.04~0.1 | 0.04 |
| 고무 on 금속 | 0.5~0.8 | — |

> [!tip] 테플론(PTFE)
> 가장 낮은 마찰계수. 캠-팔로워 접촉면, 슬라이더 가이드에 적용 권장.

## 베어링 종류

### 슬리브 베어링 (부시 베어링, Bushing)

```
   ┌────┐
   │ 축 │  ← 회전
   └────┘
   ████████  ← 부시 (슬리브)
   □□□□□□□□  ← 하우징
```

- 구조 단순, 저비용
- 저속·저하중에 적합
- 청동(bronze), 나일론, PTFE 부시 권장
- 윤활 필요

### 볼 베어링

```
   ┌───┐
   │ 축│
   └───┘
   ○○○○○○  ← 볼들 (구름 접촉)
   ┌──────┐
   │ 외륜  │
   └──────┘
```

- 낮은 마찰 (μ ≈ 0.001~0.005)
- 고속·고정밀에 적합
- 라디얼 하중 + 약간의 액시얼 하중 수용
- **회전 축에 반드시 권장**

> [!important] 프로젝트 권장
> 크랭크 피봇 등 모든 회전 지점에 볼 베어링 삽입.
> **625ZZ (내경 5mm, M5 피봇 핀과 내경 일치)** 또는 604ZZ (내경 4mm) 사용.
> ※ 6001ZZ (내경 12mm)는 M5 피봇 핀과 7mm 갭 → 사용 불가.

### 스러스트 베어링 (Thrust Bearing)

축 방향(액시얼) 하중 전담. 수직 하중이 있는 축에 추가.

## 마찰 줄이는 방법 (우선순위)

```
1순위: 볼 베어링 사용 (효과 가장 큼)
2순위: 윤활제 (그리스, 오일) 적용
3순위: PTFE(테플론) 부시 또는 테이프 적용
4순위: 접촉 면적 최소화 설계
```

---

## 캠-팔로워 마찰

캠과 팔로워의 접촉은 연속적 슬라이딩 → 마찰 큼.

**해결책:**
```
팔로워 끝에 볼 베어링 롤러 달기:
  
  팔로워 암 ─────● ← 볼 베어링 롤러
                  │
                캠 윤곽
```

→ 슬라이딩 마찰 → 구름 마찰로 전환. 마찰력 10분의 1 수준으로 감소.

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "마찰력 계산" | `F = μ × N` |
| "기동 시 더 큰 힘 필요한 이유" | 정지 마찰계수 > 운동 마찰계수 |
| "가장 낮은 마찰계수 재료" | 테플론(PTFE) |
| "고속 회전 축 지지" | 볼 베어링 |
| "캠-팔로워 마찰 최소화" | 롤러 팔로워 (볼 베어링 장착) |
