# DC 모터 (DC Motor) (★★★)

#motor #motor-dc

## Overview Table

| 항목 | 설명 |
|------|------|
| **원리** | 전류 방향에 따라 정방향/역방향 회전 |
| **제어** | H-Bridge + PWM으로 속도 및 방향 제어 |
| **종류** | 브러시드(Brushed), 브러시리스(Brushless) |
| **특징** | 연속 회전. 위치 피드백 없음 (엔코더 추가 시 가능) |
| **권장 용도** | 연속 회전 구동 (일정 속도) |

---

## DC 모터 특성 곡선

```
토크 (τ)
│ ↑ 최대 토크 (Stall Torque)
│ ╲
│  ╲
│   ╲
│    ╲ 동작점 (Operating Point)
│     ╲
│      ╲ ← 무부하 속도 (No-load Speed)
└──────────────────────────→ RPM
```

- **무부하 속도 (No-load Speed)**: 부하 없을 때 최대 회전수
- **실속 토크 (Stall Torque)**: 회전 불가 시 최대 토크 (순간적, 과열 주의)
- **최적 동작점**: 최대 효율 = Stall Torque × No-load Speed / 4

## 기어드 DC 모터 (Geared DC Motor) ★★★

> [!note] 기어드 DC 모터 특징
> 일반 DC 모터는 수천 RPM → 연속 회전 구동에는 너무 빠름
> **기어박스 내장 DC 모터** 선택: 출력 12~120 RPM
>
> 예시:
> - 12V 기어드 DC 모터, 출력 60 RPM, 토크 5 kgf·cm

## H-Bridge로 방향 제어

```
H-Bridge 구조:
         +V
          │
    S1───┤├───S2
    │    모터   │
    S3───┤├───S4
          │
         GND

정방향: S1, S4 닫힘 (S2, S3 열림) → 전류 좌→우
역방향: S2, S3 닫힘 (S1, S4 열림) → 전류 우→좌
제동:   S3, S4 동시 닫힘 → 모터 양단 GND → 역기전력으로 급제동 (Dynamic Braking)
코스팅: 모든 스위치 열림 → 관성으로 천천히 정지
주의: S1+S3 or S2+S4 동시 닫힘 → +V-GND 단락 → 드라이버 파손!
```

## L298N 모터 드라이버

가장 흔한 DC 모터 드라이버. 2채널, 각 최대 2A.

```
   Arduino          L298N
   
   PWM 핀(9) ──→ ENA (속도 제어)
   디지털(7) ──→ IN1 (방향)
   디지털(8) ──→ IN2 (방향)
   GND ──────→ GND
                │
              모터A 출력 ──→ DC 모터
                │
              12V 전원 입력
```

**코드:**
```cpp
int ENA = 9;   // PWM
int IN1 = 7;
int IN2 = 8;

void setup() {
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
}

void motorForward(int speed) {
  // speed: 0~255
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, speed);
}

void motorStop() {
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, 0);
}
```

---

## 속도 제어 (PWM)

```
analogWrite(ENA, value);
  value 0   → 정지
  value 128 → 약 50% 속도
  value 255 → 최대 속도

주의: PWM 핀만 사용 가능 (Uno: 3,5,6,9,10,11번 핀)
```

## 플라이백 다이오드 (Flyback Diode)

> [!warning] 필수 보호 회로
> 모터 정지 시 역기전력(back-EMF) 스파이크 발생 → Arduino 핀/드라이버 손상
>
> **해결**: 모터 단자 양단에 플라이백 다이오드 역방향 연결 — 범용: 1N4007 / PWM 환경: 쇼트키 **1N5819** 권장
>
> ```
>     [모터]
>   ┤+     -├
>   │       │
>   └──>|───┘  ← 다이오드 (캐소드가 + 쪽)
> ```
>
> L298N 등 대부분의 드라이버 IC는 내부 다이오드 내장 → 별도 추가 불필요

## 엔코더 (Encoder) 추가 시

위치 피드백이 필요하면 엔코더가 있는 기어드 모터 선택.

```
엔코더 A, B 채널 → Arduino 인터럽트 핀에 연결
→ 펄스 카운트 → 회전 각도 계산
→ 폐루프 위치 제어 가능
```
