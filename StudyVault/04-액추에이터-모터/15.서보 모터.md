# 서보 모터 (Servo Motor) (★★★)

#motor #motor-servo

## Overview Table

| 항목 | 설명 |
|------|------|
| **원리** | 내장 기어박스 + 포텐시오미터(위치 피드백) + 제어 회로 |
| **제어 방식** | 펄스 폭 (1~2ms, 50Hz) |
| **범위** | 표준: 0°~180°. 연속 회전 서보: 무한 회전 |
| **라이브러리** | `#include <Servo.h>` |
| **장점** | 위치 제어 쉬움, 드라이버 내장 |
| **단점** | 180° 범위 제한, 연속 회전 시 속도 정밀도 낮음 |

---

## 서보 내부 구조

```
[신호선] → [제어 회로] → [모터] → [기어박스] → [출력축]
                ↑                           │
                └──── [포텐시오미터 피드백] ─────┘
```

**신호 케이블 색상:**
- 갈색/검정: GND
- 빨강: VCC (5~6V)
- 주황/노랑/흰색: 신호 (Signal)

---

## 펄스 폭과 각도

```
신호 파형 (50Hz, 주기 20ms):

|─────────────────────────────────────|  20ms
|─────|                                  펄스폭 1ms → 0°
|─────────|                              펄스폭 1.5ms → 90°
|───────────────|                        펄스폭 2ms → 180°
```

> [!warning] 중요
> 서보 신호 ≠ `analogWrite()` PWM
> `analogWrite()`는 주파수 490Hz or 980Hz → 서보 손상 가능
> **반드시 `Servo` 라이브러리 사용!**

## Arduino 서보 제어 코드

```cpp
#include <Servo.h>

Servo legServo;       // 서보 객체 생성

void setup() {
  legServo.attach(9); // 디지털 핀 9에 연결
  // attach(pin, minPulse, maxPulse) 로 범위 보정 가능
  // legServo.attach(9, 500, 2500); // 더 넓은 범위
}

void loop() {
  legServo.write(0);    // 0도
  delay(500);
  legServo.write(45);   // 45도 (다리 들기)
  delay(300);
  legServo.write(0);    // 복귀
  delay(700);
}
```

## 서보 스펙 읽기

| 스펙 항목 | 의미 | 예시 |
|-----------|------|------|
| Stall Torque | 최대 유지 토크 | 13 kgf·cm (6V 기준) |
| Operating Speed | 60° 이동 시간 | 0.11sec/60° |
| Operating Voltage | 동작 전압 | 4.8~6.0V |
| Dead Band Width | 최소 펄스 변화 인식 범위 | 4μs |

**적합한 서보 선택 (말 다리용):**
```
필요 토크: 8 kgf·cm (계산값) × 1.5 (안전계수) = 12 kgf·cm
→ MG996R (13 kgf·cm) 또는 DS3218 (20 kgf·cm) 선택
```

## 다중 서보 제어

```cpp
#include <Servo.h>

Servo servo1, servo2, servo3, servo4; // 다리 4개

void setup() {
  servo1.attach(3);
  servo2.attach(5);
  servo3.attach(6);
  servo4.attach(9);
}

// ⚠️ delay() 방식으로는 실제 갤럽 구현 불가
//    (한 다리 처리 중 다른 다리 위상 진행 불가능)
//
// 올바른 갤럽 개념 (millis() 위상 오프셋):
//   1주기 1000ms, 각 다리 250ms 공중 / 750ms 착지
//   위상: 우뒤 0ms → 좌뒤 250ms → 우앞 500ms → 좌앞 750ms
//
// 실제 구현: [[제어 시퀀스 설계]] 참조
//   unsigned long phase = (now - gallopStart + PHASE_OFFSET[i]) % 1000;
//   if (phase < 250) servo[i].write(45);  // 공중
//   else             servo[i].write(0);   // 착지
```

> [!tip] 더 나은 방법
> `delay()` 대신 `millis()` 기반 논블로킹 → [[21.타이머와 인터럽트]]

## PCA9685 서보 드라이버 (다중 서보)

Arduino 핀 부족 시: I2C 서보 드라이버 보드 사용.

```cpp
#include <Adafruit_PWMServoDriver.h>

Adafruit_PWMServoDriver pwm = Adafruit_PWMServoDriver();

void setup() {
  pwm.begin();
  pwm.setOscillatorFrequency(27000000);
  pwm.setPWMFreq(50);  // 서보용 50Hz
}

// 채널 0~15에 서보 16개 연결 가능
void setServoAngle(uint8_t channel, uint8_t angle) {
  uint16_t pulse = map(angle, 0, 180, 150, 600);
  pwm.setPWM(channel, 0, pulse);
}
```

→ [[22.I2C와 SPI 통신]] 참조

## 서보 vs 스테퍼 비교

| 항목 | 서보 (취미용) | 스테퍼 |
|------|---------------|--------|
| 위치 제어 | 쉬움 (write(angle)) | 스텝 수 카운트 필요 |
| 드라이버 | 내장 | 외장 드라이버 필수 |
| 파워 | 중간 | 중간~높음 |
| 속도 정밀도 | 낮음 | 높음 |
| 토크 유지 | 항상 (전원 시) | 항상 (코일 통전 시) |
| 탈조 위험 | 없음 | 과부하 시 탈조 |

| 시나리오 / 키워드 | 답 |
