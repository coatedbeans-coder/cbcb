## 모터 역할별 분류

| 역할 | 권장 모터 | 이유 |
|------|-----------|------|
| **다리 4개 개별 제어** | 서보 모터 4개 | 위치 제어 쉬움, 드라이버 내장 |
| **라이더 상체 흔들림** | 서보 모터 1개 | 각도 제어 정밀 |

## 서보 방식 (채택 설계)

```
장점:
  ✅ 속도·패턴 소프트웨어로 자유롭게 변경
  ✅ 제작 간단
  ✅ 각 다리 독립 제어

단점:
  ✗ 여러 서보 동기화 → 프로그래밍 복잡
  ✗ 전류 소비 큼 (서보 4개 동시)

구성:
  Arduino
  → 서보 4개 (각 다리당 1개)
  → PWM 신호로 개별 제어
```

## 토크 계산 워크시트

```
변수:
  말 다리 조립체 총 무게 (m) = ___ g = ___ kg
  무게 중심까지 팔 길이 (r) = ___ mm = ___ m
  예상 가속도 (a) = 2~5 m/s² (보통 설계값)
  안전계수 (SF) = 2~3

계산:
  정적 토크: τ_static = m × g × r
  동적 토크: τ_dynamic = m × a × r
  총 필요 토크: τ_total = (τ_static + τ_dynamic) × SF

kgf·cm 변환: τ[kgf·cm] = τ[N·m] / 0.0981
```

**예시 계산:**
```
m = 120g = 0.12kg, r = 33mm = 0.033m, SF = 2.5
(다리 길이 D = 75mm, 크랭크 r_crank = 18mm 기준)

τ_static = 0.12 × 9.81 × 0.033 = 0.039 N·m = 0.4 kgf·cm
τ_dynamic = 0.12 × 3 × 0.033 = 0.012 N·m = 0.12 kgf·cm

τ_total = (0.039 + 0.012) × 2.5 = 0.128 N·m = 1.3 kgf·cm

→ 서보: 최소 2 kgf·cm 이상 선택 (MG90S: 2.2 kgf·cm 가능 / MG996R: 13 kgf·cm 여유 충분)
```

## 전류 소비 예상

```
서보 4개 (MG996R) 동시 구동:
  각 서보 최대 1.5A × 4 = 6A
  → 6V 10A 이상 전원 필요

Arduino + PCA9685 로직:
  USB 5V (컴퓨터 또는 USB 어댑터) → Arduino에만 공급하면 PCA9685 로직 전원 동시 공급
  ※ 서보 전원(6V 10A)과 Arduino 전원(USB 5V)은 GND만 공유하고 전원선은 분리할 것
```

## 최종 권장 설계 (초심자)

> [!tip] 가장 쉬운 시작점
>
> **서보 방식 채택:**
> - 다리 4개: MG996R 서보 × 4
> - 라이더: SG90 서보 × 2 (상체 좌우/전후)
> - 제어: Arduino Uno + PCA9685 서보 드라이버
> - 전원: 6V 10A 스위칭 어댑터 (서보용)
>         5V 2A USB 어댑터 (Arduino용)
