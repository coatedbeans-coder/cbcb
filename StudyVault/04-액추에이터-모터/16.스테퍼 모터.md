# 스테퍼 모터 (Stepper Motor) (★★)

#motor #motor-stepper

## Overview Table

| 항목 | 설명 |
|------|------|
| **원리** | 코일 순서 통전 → 1스텝씩 정밀 회전 |
| **특징** | 엔코더 없이도 위치 제어 (오픈루프) |
| **종류** | 바이폴라 (4선), 유니폴라 (5/6선) |
| **라이브러리** | `#include <Stepper.h>` 또는 AccelStepper |
| **단점** | 탈조(stall) 가능, 발열 큼, 느림 |

## 스테퍼 vs 서보 선택

| 기준 | 스테퍼 유리 | 서보 유리 |
|------|------------|----------|
| 위치 정밀도 | ✅ (스텝 단위) | △ (포텐시오미터 분해능) |
| 속도 | ✗ (느림) | ✅ (빠름) |
| 지속 토크 | ✅ | △ (온도에 따라 변함) |
| 비용 | ✅ 저렴 | △ 약간 비쌈 |
| 외장 드라이버 | 필요 ✗ | 내장 ✅ |

> [!tip] 프로젝트 선택 가이드
> - 다리 4개 순차 구동 (갤럽) → **서보** 추천 (간단, 빠름)
> - 라이더 상체 고정 자세 → **서보 or 스테퍼** 둘 다 가능

## 스텝 각도와 분해능

```
일반 스테퍼: 1.8° / step → 200 steps = 1회전
고정밀 스테퍼: 0.9° / step → 400 steps = 1회전

마이크로스텝 (A4988 드라이버):
  1/2 스텝 → 0.9° → 400 steps/rev
  1/16 스텝 → 0.1125° → 3200 steps/rev
  → 분해능 높아지나 토크 감소
     풀스텝 대비: 1/2스텝≈70%, 1/4스텝≈54%, 1/8스텝≈38%, 1/16스텝≈19%
```

## 바이폴라 스테퍼 (4선) 코드

```cpp
#include <Stepper.h>

const int STEPS_PER_REV = 200;
// 핀 순서: IN1, IN2, IN3, IN4 (L293D or ULN2003)
Stepper camStepper(STEPS_PER_REV, 2, 3, 4, 5);

void setup() {
  camStepper.setSpeed(30); // 30 RPM
}

// ⚠️ 단일 모터 테스트 전용 — 실제 작품에는 millis() 기반 논블로킹 방식 사용
void loop() {
  camStepper.step(200);    // 1바퀴 정방향
  delay(1000);
  camStepper.step(-200);   // 1바퀴 역방향
  delay(1000);
}
```

## A4988 드라이버 (권장)

NEMA 17 스테퍼에 가장 많이 사용.

```
A4988 핀 배치:
  VMOT: 모터 전원 (8~35V)
  GND:  공통 GND
  VDD:  로직 전원 (5V)
  
  STEP: 스텝 펄스 (Arduino PWM or 디지털)
  DIR:  방향 (HIGH=CW, LOW=CCW)
  
  MS1, MS2, MS3: 마이크로스텝 설정
  (000=1 스텝, 001=1/2, 010=1/4, 011=1/8, 111=1/16)
```

**A4988 코드:**
```cpp
#define STEP_PIN 3
#define DIR_PIN  4

void setup() {
  pinMode(STEP_PIN, OUTPUT);
  pinMode(DIR_PIN, OUTPUT);
  digitalWrite(DIR_PIN, HIGH); // 정방향
}

void stepMotor(int steps, int delayUs) {
  for (int i = 0; i < steps; i++) {
    digitalWrite(STEP_PIN, HIGH);
    delayMicroseconds(delayUs);
    digitalWrite(STEP_PIN, LOW);
    delayMicroseconds(delayUs);
  }
}
```

## AccelStepper 라이브러리 (가속/감속 포함)

```cpp
#include <AccelStepper.h>

AccelStepper stepper(AccelStepper::DRIVER, 3, 4);
// (드라이버 타입, STEP핀, DIR핀)

void setup() {
  stepper.setMaxSpeed(1000);    // 스텝/초
  stepper.setAcceleration(500); // 스텝/초²
  stepper.moveTo(200);          // 목표 위치 (200 스텝)
}

void loop() {
  stepper.run(); // 반드시 루프에서 반복 호출!
}
```

## 발열 관리

> [!warning] 스테퍼 발열 주의
> 스테퍼는 정지 중에도 코일 통전 → 지속 발열
>
> **해결책:**
> 1. 전류 제한: A4988의 VREF 조정으로 전류 제한
> 2. 정지 시 enable 핀 LOW (드라이버 비활성화)
> 3. 방열판 부착
> 4. 동작 듀티 사이클 낮게 유지
