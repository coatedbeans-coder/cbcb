# 🐎 말을 탄 여인 — 프로젝트 개요 (★★★)

#project

> [!important] 프로젝트 정의
> **제목**: 말을 탄 여인 (Woman Riding a Horse)
> **장르**: 메카트로닉스 키네틱 아트
> **핵심**: 갤럽(gallop) 동작하는 말 + 상체가 움직이는 여성 라이더
> **제어**: Arduino 기반 자동화

---

## 🎯 동작 목표

| 동작         | 설명              | 구현 방법   |
| ---------- | --------------- | ------- |
| 말 갤럽       | 4다리 순차 교차 운동    | 서보 4개   |
| 라이더 상체 흔들림 | 앞뒤 ± 좌우 리드미컬 스윙 | 서보 1~2개 |
| 라이더 팔 동작   | 고삐 쥔 팔 미세 움직임   | 마이크로서보  |
| 말 고개 끄덕임   | 갤럽과 동기화         | 마이크로서보  |

            │
    ┌───────▼───────┐

       │  │  │  │
    서보 서보 서보 서보

## 📐 물리적 규모 계획

```
말 크기:
  전장: 16cm
  높이: 14cm (말 등까지)
  너비: 5cm

라이더:
  높이: 10~12cm (말 등 위에서)
  재료: 조각 + 섬유 + 가죽

전체 조형물:
  베이스 포함 총 높이: 30cm (확정)
  무게: 약 0.5~1kg (목표, 할로우 바디 경량화)
```

## 🗓️ 제작 단계 (로드맵)

```
1단계: 메커니즘 프로토타입 (합판 + 임시 서보)
  → 갤럽 동작 검증
  → 타이밍 조정
  → 토크 계산 검증

2단계: 구조 제작 (알루미늄 프레임)
  → T-slot 프로파일로 말 골격
  → 베어링, 피봇 정밀 제작
  → 서보 브래킷 체결

3단계: 전자제어 시스템
  → Arduino 코드 작성
  → 서보/모터 배선
  → 전원 시스템 구성

4단계: 외형 제작 (아트)
  → 말 외형: 아크릴 3mm 레이저컷 실루엣 패널 × 2장 (할로우 바디)
  → 패널 채색 or 무도장 (투명 아크릴 그대로 메커니즘 노출)
  → 라이더 조각
  → 도색 및 마감

5단계: 통합 테스트
  → 장시간 구동 테스트 (8시간)
  → 소음/진동 점검
  → 안전 장치 확인
```

## ⚡ 핵심 설계 결정사항

### 결정 1: 구동 방식
> **선택: 서보 방식 (다리 4개 × MG996R)**
> 이유: 제작 간단, 소프트웨어 유연성, 갤럽 패턴 쉽게 변경

> [!warning] MG90S 전압 주의
> 라이더 상체 서보 MG90S 권장 전압은 **4.8~6V**이며 6V가 상한선.
> 서보 전원 6V 어댑터 사용 시 적정하나, 케이블 전압 강하 후 실측 확인 필요.

### 결정 2: 제어 방식
> **선택: Arduino Uno + PCA9685 서보 드라이버**
> 이유: 서보 채널 확장, millis() 기반 논블로킹 제어

### 결정 3: 프레임
> **선택: 합판 프로토타입 → 알루미늄 T-slot 최종**
> 이유: 프로토타입 빠른 수정 가능, 최종은 내구성
