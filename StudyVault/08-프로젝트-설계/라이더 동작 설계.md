# 라이더 동작 설계 (★★)

#project #mechanism #motor-servo

## 라이더 동작 목표

| 동작 | 주파수 | 진폭 | 구현 |
|------|--------|------|------|
| 상체 전후 흔들림 | 갤럽과 동기 (1 Hz) | ±10° | 서보 1개 |
| 상체 좌우 흔들림 | 갤럽의 1/2 (0.5 Hz) | ±5° | 서보 1개 |
| 팔(고삐) 미세 흔들림 | 0.3 Hz | ±15° | SG90 1개 |
| 고개 움직임 | 1.0 Hz (갤럽과 동기) | ±8° | SG90 1개 |

---

## 피봇 구조 설계

```
        [머리]
          │ 고개 피봇 (SG90)
        [목]
          │
        [상체]
          │ 전후 피봇 (MG90S)
          │ 좌우 피봇 (MG90S)
        [허리/안장]
          │ 고정 (말 등에 볼트)
        [말 등 프레임]
```

### 이중 피봇 구조

```
전후 피봇 (X축):
  서보A → 크랭크 암 → 상체 연결로드
  
         서보A혼
            │ ← 갤럽 동기 (1 Hz)
    ────●───●────
    │              상체
   말등           (앞뒤 ±10°)

좌우 피봇 (Y축):
  서보B → 크랭크 암 → 상체 연결로드
  → 갤럽의 절반 주파수 (0.5 Hz, ±5°)
```

## 라이더 제어 코드

```cpp
// 라이더 상체 흔들림 (사인파 생성)
// millis() % 주기로 정규화 → float 정밀도 손실 방지 (장시간 구동 안전)
void updateRider() {
  unsigned long t = millis();

  // 전후 흔들림: 갤럽 1 Hz와 동기화
  float phaseFB = (float)(t % 1000) / 1000.0;  // 0.0~1.0 정규화
  float frontBack = 90.0 + 10.0 * sin(2 * PI * phaseFB);
  riderFB.write((int)frontBack);  // 전후 서보

  // 좌우 흔들림: 0.5 Hz (2초 주기)
  float phaseLR = (float)(t % 2000) / 2000.0;
  float leftRight = 90.0 + 5.0 * sin(2 * PI * phaseLR);
  riderLR.write((int)leftRight);  // 좌우 서보

  // 고삐 팔 흔들림: 0.3 Hz
  float phaseArm = (float)(t % 3333) / 3333.0;
  float armSwing = 90.0 + 15.0 * sin(2 * PI * phaseArm);
  riderArm.write((int)armSwing);
}
```

## 라이더 조각 구조

```
제작 재료:
  - 몸체: 발포 폼 (EPP) 조각 → 에폭시 퍼티 마감
  - 팔다리: 알루미늄 와이어 코어 + 발포 폼
  - 옷: 실제 천으로 제작 (역동적 움직임 강화)
  - 머리: 점토 조각 → 에폭시 코팅

피봇 포인트:
  허리: 알루미늄 플레이트 + M4 피봇 볼트
  목: 알루미늄 혼 + M3 피봇
```
