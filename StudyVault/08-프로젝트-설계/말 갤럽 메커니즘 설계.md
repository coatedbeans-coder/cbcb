# 말 갤럽 메커니즘 설계 (★★★)

#project #mechanism

## 말의 갤럽 보행 분석

### 실제 말 갤럽 순서

```
갤럽 (4박자 순서): 우뒤 → 좌뒤 → 우앞 → 좌앞

시간 →   0%    25%   50%   75%   100%
우뒤다리  ▐████░░░░░░░░▐████░░░░░░░░
좌뒤다리  ░░░░▐████░░░░░░░░▐████░░░░
우앞다리  ░░░░░░░░▐████░░░░░░░░▐████
좌앞다리  ░░░░░░░░░░░░▐████░░░░░░░░░

▐████ = 공중 (다리 올림)
░░░░ = 착지 (다리 내림)
```

**핵심**: 각 다리가 1/4 주기(90°/25%) 씩 위상 차이. 뒷다리가 먼저 박차고 앞다리가 착지.

## 서보 방식 (채택 설계)

```
Arduino + PCA9685
  │
  ├── 서보1 (채널0) → 우앞다리 크랭크-링크
  ├── 서보2 (채널1) → 우뒤다리 크랭크-링크
  ├── 서보3 (채널2) → 좌앞다리 크랭크-링크
  └── 서보4 (채널3) → 좌뒤다리 크랭크-링크
```

**각 다리 서보 각도 계획:**
```
착지 위치: 0° (서보 0도)
공중 위치: 45° (서보 45도)

갤럽 1주기 = 1000ms

우뒤 타임라인 (기준, 0ms 시작):
  0ms:       0° 상승 시작 (착지 위치에서 바로 들기 시작)
  0~100ms:   0→45° 상승
  100~200ms: 45° 유지
  200~350ms: 45→0° 하강
  350~1000ms: 0° 유지 (착지)

좌뒤 타임라인: 위와 동일, 250ms 오프셋
우앞 타임라인: 위와 동일, 500ms 오프셋
좌앞 타임라인: 위와 동일, 750ms 오프셋
```

## 다리 링크 기하학

```
서보 혼(크랭크) → 커넥팅 로드 → 다리 섹션

            서보 혼
              ●──────────────────● 다리 상부 피봇 (어깨)
              │ 크랭크 r=18mm      │
              │커넥팅로드 L=50mm    │
              ●──────────────────● 다리 접지부 (발굽)
              다리 길이 D = 75mm (어깨 피봇 ~ 발굽)

서보 45° 회전 시 다리 스윙 각도:
  수평 변위 x = r × sin(45°) = 18 × 0.707 = 12.7mm
  스윙 각도 θ = arcsin(x / D) = arcsin(12.7 / 75)
              = arcsin(0.169) ≈ 10°

→ 자연스러운 갤럽 스윙 ✓
※ r이 작으면 스윙 부족. D=75mm 기준 r=18mm로 ~10° 확보.
```

## 제작 순서

```
1. 다리 링크 치수 계산 (Grashof 검증)
2. 다리 링크 형상 3D 프린팅 (PETG — 강도·내열 우선)
3. 피봇(M4 볼트 20mm + 624ZZ 볼 베어링) 조립
4. 서보 마운팅
5. 단일 다리 동작 테스트
6. 4다리 동기화 테스트
7. 타이밍 미세 조정
```

## 트러블슈팅 가이드

| 증상 | 원인 | 해결 |
|------|------|------|
| 다리가 갑자기 멈춤 | 서보 과부하 | 더 큰 서보 or 링크 길이 조정 |
| 갤럽이 어색함 | 타이밍 오프셋 틀림 | 위상 오프셋 값 조정 |
| 진동 심함 | 질량 불균형 | 무게 배분 재설계 |
| 링크 잠김(Lock-up) | Grashof 조건 미충족 | 링크 길이 재설계 |
