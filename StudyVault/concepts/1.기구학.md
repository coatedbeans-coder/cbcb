### 오답 메모

**1종 지레 (받침점 위치)**
- 혼동: 받침점 위치를 모름
- 핵심: 1종 지레는 힘 — 받침점 — 하중 순서. 받침점이 중간에 위치. 시소·가위가 대표 예.

**Grashof 조건 불만족 시 동작**
- 혼동: Grashof 조건 결과를 모름
- 핵심: s + l > p + q 이면 어떤 링크도 완전 회전 불가 → 전부 왕복(로커) 운동만 가능.

**캠 스트로크 계산법**
- 혼동: 스트로크 계산 방법을 모름
- 핵심: Throw = 최대 반지름 − 최소 반지름. 로브의 최고점과 최저점의 반지름 차이.

**기어비와 출력 토크 계산**
- 혼동: 공식 자체를 모름 (6회 연속 오답)
- 핵심: GR = 피동 이 수 / 구동 이 수. τ_out = τ_in × GR (곱셈). 감속(피동>구동)이면 토크 증폭. 예: GR=3, τ_in=0.8 → τ_out=2.4 N·m.

**Grashof 조건 계산 및 판별**
- 혼동: Grashof 조건 만족 시 lock-up이 발생한다고 오해
- 핵심: s+l < p+q → 조건 만족 → 최단 링크가 크랭크 역할 시 완전 회전 가능. 잠김은 s+l = p+q 경계 케이스에서만.

**더블 크랭크 조건 (최단 링크 위치)**
- 혼동: 최단 링크를 크랭크에 두면 더블 크랭크라고 오해
- 핵심: 최단 링크 = 프레임(Ground) → 더블 크랭크. 최단 링크 = 크랭크/팔로워 → 크랭크-로커. 최단 링크 = 커플러 → 더블 로커.
- **해결**: 2026-04-24 ✅

**크랭크-슬라이더 최대 속도 위치**
- 혼동: 공식을 모름 (4회 연속 오답)
- 핵심: 0°·180° (상사점·하사점)에서 슬라이더는 순간 정지 (속도=0). 크랭크 90°·270°에서 크랭크 핀이 슬라이더 방향과 완전 일치 → 최대 속도.
