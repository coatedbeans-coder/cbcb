## 프레임 설계 전략

### 스켈레탈 구조 (골격형)

```
    ┌──────────────────┐  ← 등 프레임
    │                  │
    │   ┌──────────┐   │  ← 몸통 프레임
    │   │          │   │
    └───┴──────────┴───┘
    │   │          │   │  ← 다리 마운팅 포인트
   다리 다리      다리 다리
```

- 필요한 부분만 구조 → 가벼움
- 로봇, 아트 조형물에 적합

### 모노코크 구조 (외피형)

- 외부 껍질이 구조 담당
- 복잡한 형상에 적합
- 아트 조형의 말 외형에 활용 가능 (발포 폼 + FRP)

## 말을 탄 여인 — 프레임 설계 개요

```
  [라이더 상체]
       │ 피봇
  [라이더 하체/안장]
       │ 고정
  [말 등/몸통 프레임]
    ╱        ╲
[앞다리 링크]  [뒷다리 링크]
   │          │
[앞다리 피봇]  [뒷다리 피봇]
   │          │
[서보 마운팅 브래킷 ×4]
       │
  [베이스/받침대]
```

### 주 구조 (말 몸통)

- **소재**: 알루미늄 T-slot 20×20mm 또는 30×30mm
- **형태**: 직사각형 공간 프레임
- **서보 마운팅**: 다리별 서보 브래킷을 몸통 측면에 직접 체결

### 다리 링크

- **소재**: 알루미늄 플레이트 3~5mm 또는 합판 9mm (프로토타입)
- **피봇**: M5 볼트 + 볼 베어링 + 스냅링
- **연결봉 (Coupler)**: 알루미늄 로드 or 스레디드 로드 + 턴버클 (길이 조정)

### 베이스 (받침대)

- **소재**: 두꺼운 합판 (18mm) 또는 철 플레이트
- **무게 중심**: 낮게 유지 (안정성)
- **고무 발 (Rubber Foot)**: 진동 감쇠 + 미끄럼 방지

## 단면 형상별 강성 비교

```
같은 재료량 기준 비교:

  □ 솔리드 사각형: 기준 1배
  □□ I형 단면: 약 5~8배 ← 효율 최고
  ○ 원형 튜브: 약 3~5배
  ▣ 사각 튜브: 약 3~5배 ← 실용적 선택
  — 플랫 바: 방향에 따라 크게 차이
```

> [!tip] 프로젝트 권장
> 알루미늄 T-slot 프로파일 = 내부 공간 있는 사각 튜브 + T-홈
> → 강성 + 조립 용이성 + 확장성 모두 만족

## 하중 분석

```
정적 하중:
  - 자체 무게 (말+라이더 조형물): 약 3~5 kg
  - 베이스에서 지지

동적 하중 (갤럽 시):
  - 다리 가속/감속 → 관성력
  - 진동 → 피로 하중
  - 안전계수 3~5 적용 권장
```

---

## 조립 순서 권장

```
1단계: 베이스 프레임 조립 (가장 먼저, 기준)
2단계: 서보 브래킷 ×4 제작 & 몸통 프레임에 체결
3단계: 다리 링크 제작 & 연결
4단계: 몸통/외형 장착
5단계: 라이더 상체 피봇 추가
6단계: 서보 배선 & 전선 정리
7단계: 외형 마감 (도색, 섬유 등)
```

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "같은 재료량, 최대 강성" | I형 단면 |
| "나중에 수정·확장 용이한 프레임" | T-slot 알루미늄 프로파일 |
| "무게 중심 낮추는 이유" | 안정성 (転倒 방지) |
| "진동 감쇠 방법" | 고무 패드, 댐퍼, 유연 커플링 |
| "다리 피봇 체결 부품 3개" | 볼트 + 볼 베어링 + 스냅링 |
