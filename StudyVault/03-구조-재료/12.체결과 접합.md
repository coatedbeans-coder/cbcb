## 볼트 & 너트 (Bolts & Nuts)

가장 신뢰할 수 있는 기계 체결.

### 미터 나사 규격

| 규격 | 외경 | 피치 | 용도 |
|------|------|------|------|
| M3 | 3mm | 0.5mm | 전자 부품, 서보 혼 |
| M4 | 4mm | 0.7mm | 소형 구조물 |
| M5 | 5mm | 0.8mm | 중형 구조물 |
| M6 | 6mm | 1.0mm | 주 구조 프레임 |
| M8 | 8mm | 1.25mm | 대형 하중 |

> [!tip] 말 프레임 권장
> 주 구조: M6 볼트 + 너트 + 와셔
> 링크 피봇: M4 볼트 20mm (624ZZ 내경 4mm 일치)
> 서보 마운팅: M3 (서보 스펙 확인)

### 체결 강도 높이는 방법

```
1. 스프링 와셔 (Spring Washer) 추가 → 진동으로 인한 풀림 방지
2. 록타이트 (Loctite) 중간 강도 (파란색) 도포 → 진동 방지 + 재조립 가능
3. 플랜지 너트 사용 → 와셔 일체형, 진동 강화
4. 나일론 인서트 록너트 (Nylock) → 나일론으로 자가 잠금
```

## 접착제 종류

### 에폭시 (Epoxy)
- 2액형 (수지 + 경화제) 혼합
- 금속, 플라스틱, 목재 등 거의 모든 재료 접착
- 24시간 경화 (5분용도 있음, 강도는 낮음)
- 전단력 강함, 박리력 약함

> [!warning] 에폭시 + 반복 진동 환경
> 갤럽처럼 반복 하중이 있는 부위에 에폭시 단독 사용 금지.
> 진동 피로로 접합 면이 장기간 후 박리될 수 있음.
> → **볼트 체결이 주(主), 에폭시는 보조**로만 사용할 것.

### 순간접착제 (CA / Super Glue)
- 빠른 경화 (30초~3분)
- 목재, 플라스틱에 효과적
- 충격에 약함
- 베이킹 소다로 빠른 경화 촉진 가능

### 글루건 (Hot Glue)
- 빠른 작업, 분리 가능
- 강도 약하고 열에 약함 (60°C 이상 녹음)
- **전시 작품에는 구조 체결에 사용 금지**
- 임시 고정·프로토타입에만 사용

### UV 수지 (UV Resin)
- UV 램프로 즉시 경화
- 정밀 작업에 유용

## 핀 & 클립

```
핀 (Pin):
  - 분할핀 (Cotter Pin): 축 고정, 분리 가능
  - 스프링 핀 (Roll Pin): 압입 고정
  - 다월 핀 (Dowel Pin): 정밀 위치 결정

스냅링 / 리테이닝 링:
  - 축 방향 이동 방지
  - 베어링 고정에 필수
```

---

## 구멍 뚫기 & 나사 탭핑

```
나사 탭핑 순서:
1. 드릴로 탭 하부 구멍 뚫기
   (M3: 2.5mm, M4: 3.3mm, M5: 4.2mm, M6: 5.0mm 드릴)
2. 탭 드라이버로 나사산 내기
3. 절삭유 사용 (알루미늄: 윤활제, 강철: 절삭유)
4. 탭 반 바퀴마다 역방향으로 칩 제거
```

> [!tip] 목재 나사산
> 목재에 탭핑 불가 → 금속 나사 인서트(brass insert) 열로 압입하거나, T-nut 사용.

## 영구 접합 vs 비영구 접합 선택

```
비영구 체결 우선 원칙:
  → 유지보수, 교체, 조정이 필요한 부품은 반드시 볼트 체결
  → 캠, 링크, 모터 마운트: 비영구
  → 장식 패널, 외장: 영구 접착 가능
```

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "진동 환경에서 볼트 풀림 방지" | 록타이트 or 스프링 와셔 or Nylock 너트 |
| "전시 작품 구조 체결" | 볼트 & 너트 (글루건 금지) |
| "금속-금속 영구 접합" | 에폭시 or 용접 |
| "M5 탭핑용 드릴 직경" | 4.2mm |
| "축 방향 이동 방지" | 스냅링(리테이닝 링) |
