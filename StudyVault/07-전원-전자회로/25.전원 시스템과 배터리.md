## 말을 탄 여인 전원 설계

> [!important] 전원 분리 원칙
> **모터 전원과 Arduino/서보 전원을 반드시 분리!**
> GND는 반드시 공통 연결 (공통 기준점).
>
> ```
>         [벽 콘센트]
>              │
>      [6V / 5A 스위칭 어댑터]
>              │
>         [서보 × 8 (MG90S×6, SG90×2)]
>              │
>      [Arduino] (USB 5V 별도 공급)
>         └─── GND 공통 연결 ────┘
> ```

## 전압 / 전류 계획

```
부하 분석:
  서보 MG90S × 6:  각 0.7A → 합계 4.2A (최대 동시)
  서보 SG90   × 2:  각 0.5A → 합계 1.0A
  Arduino Uno:       0.05A
  합계 최대: 약 5.2A

전원 규격 선택:
  서보용: 6V / 5A 스위칭 어댑터
  Arduino: USB 5V (컴퓨터 또는 USB 어댑터)
  
실제 소비는 최대치보다 낮음 → 여유있게 선택
```

## 전압 강하 (Voltage Sag) 방지

```
문제:
  모터 기동 시 전류 급증 → 전압 강하 → Arduino 리셋

해결:
  1. 모터 전원 분리 (가장 효과적)
  2. 전원 라인에 전해 커패시터 추가:
     → 모터 전원 단자에 1000μF / 25V 커패시터
     → 에너지 버퍼 역할 → 전압 안정화
  3. 충분한 전선 굵기 사용 (AWG 20 이상 for 5A)
```

## 배터리 사용 시 (이동 전시)

### LiPo 배터리 주의사항

> [!warning] LiPo 안전 수칙
> - 과방전 금지: 셀당 3.0V 이하 → 영구 손상
> - 과충전 금지: 셀당 4.2V 이상 → 화재 위험
> - 단락 금지: 폭발/화재 위험
> - 충전 중 자리 비우지 말 것
> - 전용 LiPo 충전기 사용 (밸런스 충전)
> - 부풀어 오른 LiPo → 즉시 폐기

**BMS (Battery Management System):**
리튬 배터리 팩에는 보호회로(BMS) 내장 모델 선택 → 과방전/과충전 자동 차단.

### 배터리 용량 계산

```
런타임 = 배터리 용량 (mAh) / 총 전류 소비 (mA)

예시:
  배터리: 5000mAh (5Ah)
  총 소비: 2A (평균)
  런타임 = 5000mAh / 2000mA = 2.5시간
  
안전계수 70%: 실제 런타임 ≈ 1.75시간
```

## 전압 레귤레이터

| 타입 | 예시 | 특징 |
|------|------|------|
| 리니어 (LDO) | LM7805, AMS1117 | 간단, 발열 큼 |
| 스위칭 (SMPS) | LM2596, XL4016 | 효율 높음, 발열 적음 |
| 벅 컨버터 | 기성 모듈 | 참고 — 12V→6V, 12V→5V 등에 활용 |

**이 프로젝트 전원 구성:**
```
[6V / 5A 스위칭 어댑터] → 서보 전원 (MG90S × 6 + SG90 × 2)
[USB 5V]               → Arduino 전원
GND 공통 연결
```

## 디커플링 커패시터

> [!tip] 노이즈 제거 필수
> 각 모터 드라이버, 서보 드라이버 근처에 배치.
>
> - **전해 커패시터**: 1000μF / 25V → 큰 전압 변동 흡수
> - **세라믹 커패시터**: 100nF → 고주파 노이즈 제거
> - 모터 단자에 0.1μF 세라믹 커패시터 병렬 연결 → EMI 감소

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "Arduino 리셋 반복 (모터 사용 중)" | 전원 분리 + 1000μF 커패시터 추가 |
| "LiPo 최소 방전 전압" | 셀당 3.0V |
| "12V→6V 변환, 고효율" | 벅 컨버터 (스위칭 레귤레이터) |
| "서보 MG90S 6개 × 0.7A + SG90 2개 × 0.5A → 필요 전류" | 최소 5A (6V/5A 어댑터) |
| "GND 공통 연결 이유" | 기준 전위 통일 (없으면 오작동) |
