# 모터 드라이버와 H-Bridge (★★★)

#power #motor

## Overview Table

| 드라이버 | 최대 전류 | 전압 | 특징 |
|----------|-----------|------|------|
| L293D | 0.6A/채널 | 5~36V | 소형 DC 모터 |
| **L298N** | **2A/채널** | **5~35V** | **권장: 일반 DC 모터** |
| DRV8833 | 1.5A/채널 | 2.7~10.8V | 저전압, 소형 |
| TB6612 | 1.2A/채널 | 2.5~13.5V | L298N 대안, 효율 좋음 |
| IBT-2 | 43A/채널 | 5~27V | 대형 모터 |

## H-Bridge 동작 원리

```
        +V
         │
    S1──┤├──S2
    │   모터   │
    S3──┤├──S4
         │
        GND

정방향: S1(닫힘), S4(닫힘), S2(열림), S3(열림)
  전류: +V → S1 → 모터(+) → 모터(-) → S4 → GND

역방향: S2(닫힘), S3(닫힘), S1(열림), S4(열림)
  전류: +V → S2 → 모터(-) → 모터(+) → S3 → GND

제동(Brake): S3, S4 동시 닫힘 → 모터 양단 GND → 역기전력으로 급정지
             (또는 S1, S2 동시 닫힘 → 모터 양단 +V)

주의: S1+S3 or S2+S4 동시 닫힘 → 같은 암(arm) 단락 → +V-GND 직접 연결 → 드라이버 파손!
```

## L298N 연결도

```
Arduino 핀     L298N                  모터/전원
─────────────────────────────────────────────────
PWM(9)   ──→  ENA            (채널A 속도)
D7       ──→  IN1             (채널A 방향1)
D8       ──→  IN2             (채널A 방향2)

PWM(10)  ──→  ENB            (채널B 속도)
D11      ──→  IN3             (채널B 방향1)
D12      ──→  IN4             (채널B 방향2)

5V       ──→  +5V (로직전원)
GND      ──→  GND
              12V 단자 ──→  12V 전원 (모터용)
              OUT1, OUT2 ──→ 모터A
              OUT3, OUT4 ──→ 모터B
```

**코드:**
```cpp
// 채널A (기어드 DC 모터 → 캠샤프트)
int ENA = 9, IN1 = 7, IN2 = 8;

void setup() {
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
  
  // 정방향 시작
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, 180); // 약 70% 속도
}

void setSpeed(int speed) {
  // speed: -255(역방향 최대) ~ 0(정지) ~ 255(정방향 최대)
  if (speed > 0) {
    digitalWrite(IN1, HIGH);
    digitalWrite(IN2, LOW);
    analogWrite(ENA, speed);
  } else if (speed < 0) {
    digitalWrite(IN1, LOW);
    digitalWrite(IN2, HIGH);
    analogWrite(ENA, -speed);
  } else {
    // 정지
    digitalWrite(IN1, LOW);
    digitalWrite(IN2, LOW);
    analogWrite(ENA, 0);
  }
}
```

## 플라이백 다이오드 (Flyback Diode)

> [!warning] 역기전력(Back-EMF) 보호 필수
>
> **문제**: 모터 코일은 인덕터 → 전원 차단 순간 역기전력 스파이크 발생
> → 수백V 순간 전압 → Arduino/드라이버 핀 소손
>
> **해결**: 모터 양단에 플라이백 다이오드 역방향 연결 — 범용: 1N4007 / PWM·서보 고속 환경: 쇼트키 **1N5819** 권장
>
> ```
>    +12V
>     │
>     ├────┤>|────┐
>     │   다이오드  │
>    [모터]       │
>     │           │
>     └───────────┘
>    GND
>    (다이오드 캐소드가 + 쪽)
> ```
>
> L298N, TB6612 등 대부분 드라이버 IC → 내부 다이오드 내장 → 별도 추가 불필요

## 모터 전류 제한

모터 과전류 → 드라이버 소손 → 퓨즈로 보호.

```
회로 구성:
  전원 → [퓨즈] → 드라이버 → 모터
  
퓨즈 용량: 모터 정격 전류 × 2
예: 1A 모터 → 2A 퓨즈
```

## 소음 감소 (PWM 주파수)

L298N 기본 PWM 주파수 (Arduino `analogWrite()`) = 490Hz 또는 980Hz → 가청 주파수 → 모터 소음.

```cpp
// Timer2 주파수 변경으로 소음 감소 (핀 3, 11 전용)
TCCR2B = TCCR2B & 0b11111000 | 0x01; // 31.4 kHz → 가청주파수 이상
```

> [!warning] 타이머 변경 주의
> Timer0 변경 시 delay(), millis() 영향받음. 반드시 Timer2 또는 Timer1 사용.

## Exam/Test Patterns

| 시나리오 / 키워드 | 답 |
|-------------------|----|
| "DC 모터 방향 제어 IC" | H-Bridge (L298N 등) |
| "L298N 최대 전류" | 채널당 2A |
| "역기전력 보호 부품" | 플라이백 다이오드 (1N4007 범용 / 1N5819 PWM 환경 권장) |
| "IN1=HIGH, IN2=LOW" | 정방향 회전 |
| "IN1=HIGH, IN2=HIGH (L298N 기준)" | 제동 (Brake) — 다른 H-Bridge 구현에서는 단락 위험 가능 |
| "모터 소음 줄이기" | PWM 주파수 20kHz 이상으로 변경 |
